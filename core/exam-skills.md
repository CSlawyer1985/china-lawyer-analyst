## 一、审题技巧

### 1.1 三步审题法

#### 第一步：圈出关键词

**需要圈出的关键词**：

| 关键词类型 | 示例 | 说明 |
|-----------|------|------|
| 主体 | 谁？（赵某 vs 周某 vs 李某） | 明确行为主体 |
| 行为 | 什么行为？（索债 vs 绑架 | 明确行为性质 |
| 时间 | 何时？（3月1日 vs 3月8日） | 时间顺序很重要 |
| 数字 | 金额？（18万元 vs 60万元） | 数值影响法律适用 |
| 程序 | 如何处理？（复议、上诉、申诉） | 程序要求 |
| 问题类型 | 是什么？为什么？如何？ | 明确答题要求 |

**示例**：
> 赵某从被害人处得知**钱在钱某处**，管钱某要钱，**钱某不给**。赵某就雇了**李某和周某**去控制**赵某的儿子**，进而要钱某返还这**18万元钱**。

**圈出关键词**：
- 赵某（主体1）
- 李某和周某（主体2、3）
- 赵某的儿子（被害人）
- 18万元（金额）
- 控制（行为）

#### 第二步：识别问题的核心

**问题类型识别**：

| 问题类型 | 关键词 | 答题重点 |
|---------|--------|----------|
| 定性问题 | "构成什么罪""是什么行为" | 罪名/行为性质 |
| 定量问题 | "如何处罚""量刑" | 刑罚幅度、情节 |
| 程序问题 | "如何审理""是否合法" | 程序要求、合法性 |
| 效力问题 | "是否有效""是否发生效力" | 效力判断标准 |
| 责任问题 | "应承担什么责任" | 责任类型、归责原则 |

**示例**：
> "赵某索债行为：为索取非法债务控制钱小小，缺乏非法占有目的，仅成立非法拘禁罪。"

**问题类型**：定性问题
**答题重点**：罪名+构成要件分析

#### 第三步：检查是否答非所问

**自检清单**：

- [ ] 我回答的是问题问的吗？
- [ ] 我是否把"赵某"答成了"周某"？
- [ ] 我是否把"担保合同"答成了"买卖合同"？
- [ ] 我是否回答了"如何处罚"而非"构成什么罪"？

**示例错误**：
- 问题："赵某索债行为构成何罪？"
- 错误回答："周某构成绑架罪。" ❌（答非所问）
- 正确回答："赵某构成非法拘禁罪。" ✅

### 1.2 常见审题错误类型

#### 错误类型1：主体混淆

**错误示例**：
> 问题2：结合事实四，对于事实二，周某应该如何处理？
>
> 我的回答：关于周某，构成绑架罪（致人重伤）...
>
> 标准答案：赵某索债行为...仅成立非法拘禁罪。

**错误原因**：
- 问题问的是"赵某索债行为"
- 我答成了"周某应该如何处理"
- 致命的审题错误

**预防方法**：
1. 圈出问题的主体："赵某"
2. 确认回答的主体："赵某"（而非"周某"）
3. 答完后重新看一遍问题

#### 错误类型2：行为混淆

**错误示例**：
> 问题：担保合同是否对甲公司发生效力？
>
> 我的回答：买卖合同对甲公司发生效力...
>
> 标准答案：担保合同对甲公司不发生效力。

**错误原因**：
- 问题问的是"担保合同"
- 我答成了"买卖合同"
- 混淆了两个不同的合同

**预防方法**：
1. 圈出问题的行为："担保合同"
2. 确认分析的是"担保合同"（而非"买卖合同"）
3. 区分案件中的多个行为

#### 错误类型3：问题类型混淆

**错误示例**：
> 问题：C和E的股权转让是否有效？
>
> 我的回答：股权转让后需要办理登记...
>
> 标准答案：股权转让有效。

**错误原因**：
- 问题问的是"是否有效"（效力问题）
- 我回答的是"需要办理登记"（程序问题）
- 虽然相关，但不是问题的核心

**预防方法**：
1. 识别问题类型："是否有效"=效力问题
2. 直接回答："有效"或"无效"
3. 再分析理由

### 1.3 分层审题法

#### 适用场景
- 复杂案情
- 多个问题
- 多个法律关系

#### 分层方法

**第一层：整体把握**
- 通读全部案情
- 了解时间线
- 识别所有主体
- 明确所有法律关系

**第二层：分段标记**
```
事实一：钱某出借银行卡 → 诈骗共犯 → 占有赃款18万元
事实二：赵某指使李某、周某绑架钱小小 → 索要18万元 → 钱小小重伤
事实三：赵某抢夺 → 转化为抢劫 → 致刘某轻伤
事实四：投案自首 → 争议问题澄清
```

**第三层：对应问题**
- 问题1：对应事实一
- 问题2：对应事实二
- 问题3：对应事实二+四
- 问题4：对应事实三
- 问题5：对应事实四

## 二、答题技巧

### 2.1 IRAC答题法

#### 什么是IRAC

**IRAC** = Issue + Rule + Application + Conclusion
- **Issue（问题）**：将法律问题具体化
- **Rule（规则）**：识别适用法律规则
- **Application（适用）**：将事实与法律规则匹配
- **Conclusion（结论）**：得出明确结论

#### 示例：担保合同效力

**Issue（问题）**：
> G未出具董事会决议即签订担保合同，该担保合同是否对甲公司发生效力？

**Rule（规则）**：
> 《民法典担保制度解释》第7条：公司的法定代表人违反公司法关于公司对外担保决议程序的规定，超越权限代表公司与相对人订立担保合同，相对人非善意的，担保合同对公司不发生效力。
>
> 善意的判断标准：相对人根据公司公开披露的信息审查决议，未审查决议的，应当认定其为非善意。

**Application（适用）**：
1. G超越权限：未出具董事会决议即签订担保合同 ✅
2. 相对人乙公司：未审查董事会决议 ✅
3. 乙公司非善意 ✅

**Conclusion（结论）**：
> 担保合同对甲公司不发生效力。

### 2.2 三段论答题法

#### 什么是三段论

**三段论** = 大前提 + 小前提 + 结论
- **大前提**：法律规则
- **小前提**：案件事实
- **结论**：法律适用结果

#### 示例：索债行为定性

**大前提（法律规则）**：
> 《刑法》第238条第3款：为索取债务非法拘禁他人的，依照非法拘禁罪的规定处罚。
>
> 最高人民法院解释：索取高利贷、赌债等法律不予保护的债务的，也以非法拘禁罪定罪处罚。

**小前提（案件事实）**：
- 赵某为索要18万元（实为诈骗赃款）
- 主观上认为是"索取债务"
- 非法拘禁钱小小

**结论**：
> 赵某构成非法拘禁罪，不构成绑架罪。

---

### 2.3 关键词答题法

#### 什么是关键词答题法

**核心思想**：在答案中包含评分的关键词

#### 示例：认罪认罚从宽

**问题**：对郑某能否适用认罪认罚从宽制度？

**关键词**：
- 认罪认罚从宽制度
- 《关于适用认罪认罚从宽制度的指导意见》第6条
- 犯数罪
- 仅如实供述其中一罪
- 全案不作"认罪"的认定
- 不适用认罪认罚从宽制度

**答案模板**：
> 根据《关于适用认罪认罚从宽制度的指导意见》第6条的规定，犯罪嫌疑人、被告人犯数罪，仅如实供述其中一罪或部分罪名事实的，全案不作"认罪"的认定，不适用认罪认罚从宽制度。
>
> 本案中，郑某对危险驾驶罪认罪认罚，对妨害公务罪不认罪，属于部分认罪，全案不适用认罪认罚从宽制度。

### 2.4 多角度论证法

#### 适用场景
- 需要从多个角度论证同一结论
- 增强说服力
- 展示思维的全面性

#### 示例：责令临时停业的性质

**角度1：行为特征**
> 责令临时停业是对生产经营活动的暂时性限制，符合行政强制措施的特征（暂时性、预防性）。

**角度2：与后续处罚的关系**
> 应急管理局后续作出的"责令停业+罚款7万元"属于行政处罚，是对公司"逾期未取得许可证"这一持续违法行为的终局制裁。若前期临时停业属于行政处罚，则违反"一事不再罚"原则，进一步说明其行政强制措施的属性。

**角度3：法律规定**
> 《行政强制法》第2条：行政强制措施是暂时性控制。责令临时停业符合这一特征。

**结论**：
> 综合以上三个角度，责令临时停业属于行政强制措施。

## 三、常见答题错误

### 错误1：结论不明确

**错误示例**：
> 担保合同可能对甲公司发生效力，也可能不发生效力，需要进一步分析...

**正确示例**：
> 担保合同对甲公司不发生效力。理由如下...

**改进方法**：
- 第一句话就明确结论
- 不要模糊、含蓄
- 结论+理由的结构

### 错误2：理由不充分

**错误示例**：
> 担保合同对甲公司不发生效力，因为G没有经过董事会批准。

**正确示例**：
> 担保合同对甲公司不发生效力。理由：
> 1. 根据《民法典担保制度解释》第7条，相对人未审查决议的，担保合同对公司不发生效力。
> 2. 本案中，乙公司未审查甲公司董事会决议，未尽到合理注意义务。
> 3. 因此，乙公司非善意，担保合同对甲公司不发生效力。

**改进方法**：
- 引用具体法律条文
- 逐一分析构成要件
- 充分论证

### 错误3：遗漏关键要点

**错误示例**：
> 赵某构成绑架罪。

**正确示例**：
> 赵某构成非法拘禁罪。理由：
> 1. 赵某为索要18万元（虽为诈骗赃款，但赵某认为是债务）
> 2. 根据《刑法》第238条第3款，为索取债务非法拘禁他人的，定非法拘禁罪
> 3. 赵某缺乏非法占有目的（区别于绑架罪的勒索）
> 4. 因此构成非法拘禁罪，不构成绑架罪。

**改进方法**：
- 区分相似罪名（绑架 vs 非法拘禁）
- 分析主观目的（勒索 vs 索债）
- 引用法律依据

## 四、答题模板

### 模板1：定性问题

```
【结论】
XXXX行为构成XXXX罪/XXXX合同有效/XXXX行为违法。

【法律依据】
《XXXX法》第XX条：...

【分析】
1. XXXX行为的特征：...
2. 符合XXXX罪的构成要件：...
   - 主观方面：...
   - 客观方面：...
   - 主体：...
   - 客体：...
3. 不构成XXXX罪/不适用XXXX规则：...

【结论】
综上，XXXX行为构成XXXX罪。
```

### 模板2：效力问题

```
【结论】
XXXX合同对XX公司发生效力/不发生效力。

【法律依据】
《XXXX法》第XX条：...

【分析】
1. 行为性质：...（是否越权、是否违法）
2. 相对人善意性：
   - 善意的判断标准：...
   - 本案情况：...
   - 相对人是否善意：...
3. 法律适用：...

【结论】
综上，XXXX合同对XX公司不发生效力。
```

### 模板3：程序问题

```
【结论】
XXXX程序合法/不合法。

【法律依据】
《XXXX法》第XX条：...

【分析】
1. 程序要求：...
2. 本案情况：...
3. 是否符合程序要求：...
4. 法律后果：...

【结论】
综上，XXXX程序合法/不合法。
```

## 五、练习题

### 题目1：审题练习

**案情**：
甲公司有A、B、C、D、E、F股东，A、B、G是董事会成员，A是董事长，G是法定代表人。公司章程规定，公司对外提供担保需要经过董事会批准。G没有出具任何决议以甲公司名义对外签订了两份合同：向乙公司提供对丙公司20万债务的担保；与D签订60万的原材料买卖合同。

**问题1**：担保合同是否对甲公司发生效力？
**问题2**：买卖合同是否对甲公司发生效力？

**审题要点**：
- 圈出主体：甲公司、G、乙公司、D
- 圈出合同：担保合同（20万）、买卖合同（60万）
- 圈出问题：效力问题
- 区分两个合同的主体不同（乙公司 vs D）

### 题目2：答题练习

**问题**：赵某为索要非法债务拘禁被害人，构成何罪？

**答案**：
> 赵某构成非法拘禁罪。
>
> 理由：
> 1. 根据《刑法》第238条第3款，为索取债务非法拘禁他人的，依照非法拘禁罪定罪处罚。
> 2. 最高人民法院解释明确，索取高利贷、赌债等法律不予保护的债务的，也以非法拘禁罪定罪处罚。
> 3. 本案中，赵某为索要债务（虽为非法债务）非法拘禁他人，缺乏非法占有目的（区别于绑架罪的勒索）。
> 4. 因此构成非法拘禁罪，不构成绑架罪。
