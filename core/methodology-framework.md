# 核心方法论框架 (Core Methodology Framework)

> **来源**：最高人民法院民法典合同编通则司法解释理解与适用（代序文章）
> **标题**：合同法律适用应当把握的思维方法
> **版本**：v1.0.0
> **创建日期**：2026年2月2日
> **作者**：陈石律师（基于最高人民法院民二庭、研究室编著）

---

## 📋 框架概述

本框架系统化集成了合同法适用的**四大核心思维方法**，是确保《民法典》得到正确实施的"渔"而非"鱼"。这些思维方法不仅适用于合同纠纷，也适用于所有民商事案件的审理和分析。

**四大思维方法**：
1. **历史思维**：把握制度发展脉络，保持司法政策延续性
2. **体系思维**：运用系统观念，避免"只见树木不见森林"
3. **辩证思维**：透过现象看本质，把握量变到质变的规律
4. **程序思维**：实质性化解纠纷，避免程序空转

## 一、历史思维 (Historical Thinking)

### 1.1 核心要义

> "我国民法是伴随改革开放和社会主义市场经济体制的建立而形成并不断发展和完善的。因此，对于《民法典》合同编的具体制度和规定，也必须将其放到历史发展的脉络中进行理解，才能获得更为清晰的思路，从而确保适用的正确性。"

**基本要求**：
- ✅ 保持司法政策的延续性
- ✅ 守正与创新并举
- ✅ 将制度置于历史发展脉络中理解

### 1.2 应用指引

#### 旧司法解释效力判断清单

在判断旧司法解释是否继续适用时：

- [ ] **Step 1**: 检查旧司法解释是否与《民法典》存在冲突
  - 存在冲突 → 废止或修改
  - 无冲突且仍行之有效 → 保留或适当修改后保留

- [ ] **Step 2**: 检查司法政策性文件的规定是否可上升为司法解释
  - 《民商审判会议纪要》《民商事合同指导意见》等
  - 被证明既符合《民法典》精神又切实可行 → 上升为司法解释

- [ ] **Step 3**: 根据新的社会形势和理论成果对制度进行发展
  - "法与时转则治"
  - 更有针对性地解决当前司法实践中存在的问题

#### 司法政策延续性审查表

| 制度/规则 | 旧法规定 | 新法规定 | 司法解释演进 | 适用建议 |
|----------|---------|---------|-------------|----------|
| 违法无效认定 | 《经济合同法》（1981）：违反法律和国家政策、计划 → 无效 | 《合同法》（1999）：违反法律、行政法规强制性规定 → 无效 | 《合同法司法解释（二）》第14条：限制在"效力性强制规定" | 审查强制性规定但书的5种情形 |
| 预约合同 | 未明确规定 | 《民法典》第495条：预约合同明确约定 | 发展出：区分预约与本约、违反预约的认定、赔偿责任 | 综合交易成熟度确定赔偿范围 |
| 违约方解除 | 未明确规定 | 《民法典》第580条第2款：违约方可请求终止合同 | 《民商审判会议纪要》第48条：合同僵局可解除 | 审查是否发生合同僵局 |

#### 守正创新平衡决策树

```
问题：是否应继续沿用旧司法解释或司法政策？
│
├─ 情形A：与《民法典》存在冲突
│   └─ ❌ 不适用，应适用《民法典》及新司法解释
│
├─ 情形B：与《民法典》无冲突且仍行之有效
│   └─ ✅ 适用，保持司法政策延续性
│
└─ 情形C：与《民法典》无冲突但存在新的社会形势
    └─ ⚠️ 修订后适用，守正与创新并举
```

### 1.3 典型实务案例

#### 案例1：合同违法无效的认定（历史演进）

**制度演进脉络**：
1. **1981年《经济合同法》**：违反法律和国家政策、计划 → 无效（大量合同无效）
2. **1986年《民法通则》**：违反法律或者社会公共利益 → 无效（"法律"范围仍很广）
3. **1999年《合同法》**：限制在"违反法律、行政法规的强制性规定"
4. **《合同法司法解释（二）》第14条**：进一步限制在"效力性强制规定"
5. **2017年《民法总则》第153条第1款**：增加但书"该强制性规定不导致该民事法律行为无效的除外"
6. **《民法典》继受**：同《民法总则》第153条第1款

**历史思维的应用**：
- ✅ **守正**：继续限制合同无效的范围（避免大量合同无效）
- ✅ **创新**：直接针对《民法典》第153条第1款的但书作出解释（不再采用"效力性强制规定"表述）

**实务要点**：见《民法典合同编通则解释》第16条第1款（强制性规定但书的5种情形）

#### 案例2：未经批准合同的效力（认识深化）

**司法解释沿革**：
1. **《合同法司法解释（一）》第9条**：未经批准 → 未生效（区别于无效）
2. **《合同法司法解释（二）》第8条**：相对人可自行办理报批手续
3. **《外商投资企业规定（一）》**："两步走"方案（报批义务条款独立生效+违反整个合同违约责任）
4. **《民商审判会议纪要》**：一般化上述特殊规定
5. **《民法典》第502条**：继受上述共识

**历史思维的应用**：
- ✅ 逐步完善合同未生效向合同生效的转化机制
- ✅ 强化报批义务人的责任

**实务要点**：
- 报批义务人怠于履行报批义务 → 相对人可直接请求解除合同并赔偿损失
- 赔偿范围：参照违反预约合同的违约责任计算

#### 案例3：违约方请求终止合同（共识形成）

**争议背景**：
- 守约方故意不主动请求解除合同 → 制造或扩大违约方损失 → 合同僵局

**司法政策演进**：
1. **《合同法》**：未明确规定
2. **《民商审判会议纪要》第48条**：违约方可请求解除合同
3. **《民法典》第580条第2款**：违约方可请求终止合同权利义务关系

**历史思维的应用**：
- ✅ 《民法典》施行后，《民商审判会议纪要》第48条**并未废止**
- ✅ 在金钱债务发生合同僵局时，仍可作为裁判说理依据

**实务要点**：
- 合同终止时间：原则上以**起诉状副本送达对方的时间**为准
- 不影响违约责任的承担

### 1.4 操作建议

1. **审查旧司法解释效力时**：
   - 优先查阅《民法典时间效力规定》
   - 检查新司法解释是否废止了旧司法解释
   - 无冲突的旧司法解释可继续适用

2. **适用司法政策性文件时**：
   - 《民商审判会议纪要》《民商事合同指导意见》等
   - 与《民法典》精神无冲突 → 可作为裁判说理依据
   - 与《民法典》冲突 → 不适用

3. **处理新型疑难案件时**：
   - 回顾制度发展脉络
   - 守正：保持司法政策延续性
   - 创新：根据新形势、新成果发展制度

## 二、体系思维 (Systematic Thinking)

### 2.1 核心要义

> "要坚持系统观念，'万事万物是相互联系、相互依存的。只有用普遍联系的、全面系统的、发展变化的观点观察事物，才能把握事物发展规律。'"

**基本要求**：
- ✅ 准确把握《民法典》各编之间关系
- ✅ 全面认识各编的衔接配合关系
- ✅ 处理好《民法典》与商事特别法的关系
- ✅ 处理好《民法典》与其他法律尤其是公法规范的关系
- ✅ "举重以明轻"，避免"只见树木不见森林"

### 2.2 应用指引

#### 民法典体系关系图

```
《民法典》总分式立法体例
│
├─ 总则编（提取公因式的一般性规则）
│   ├─ 基本规定、自然人、法人、非法人组织
│   ├─ 民事权利、民事法律行为、代理
│   └─ 民事责任、诉讼时效、期间计算
│
└─ 分则各编（特殊规则）
    ├─ 物权编（物权变动、所有权、用益物权、担保物权）
    ├─ 合同编（通则发挥债法总则功能）
    ├─ 人格权编
    ├─ 婚姻家庭编
    ├─ 继承编
    └─ 侵权责任编
```

**关键关系**：
- **合同编通则**：发挥**债法总则**的功能作用
- **合同之债以外的其他债权债务关系**：同样适用合同编通则规定

#### 商事特别法与民法典关系表

| 关系类型 | 适用规则 | 典型情形 |
|---------|---------|----------|
| **特别法优先** | 商事特别法有特别规定 → 优先适用 | 公司法、证券法、保险法、票据法等 |
| **一般法补充** | 商事特别法无规定 → 适用民法典 | 公司对外担保（民法典+公司法） |
| **体系逆反纠正** | "举重以明轻"避免轻重失衡 | 侵害优先购买权合同效力（见下） |

#### 体系逆反识别与纠正清单

**识别方法**：文义解释结果是否符合体系解释要求？

**典型情形：侵害优先购买权所订合同效力**

- **文义解释**：侵害股东/共有人优先购买权 → 合同无效（错误！）
- **体系解释**："举重以明轻"
  - 无权处分（出卖人无权）→ 合同有效（仅物权不发生变动）
  - 侵害优先购买权（出卖人有权）→ 合同更应有效
- **纠正后解释**：合同不因侵害优先购买权而无效，但其他股东/共有人可主张股权/共有份额的变动无效

**类似情形**：
- 限制民事权利的"不得""必须""应当" → 不适用《民法典》第153条第1款
  - 《物权法》第191条"未经抵押权人同意，不得转让抵押财产"
    → 应根据**无权处分规则**认定合同效力
  - 《公司法》第16条限制法定代表人代表权
    → 应根据**越权代表或表见代表规则**认定合同效力

### 2.3 典型实务案例

#### 案例1：预约合同的认定与救济

**体系定位**：预约合同是交易阶段化的产物

**成立条件（与意向书的区别）**：
- ✅ 具备合同的一般成立要件
- ✅ 客体：在将来一定期限内订立本约合同
- ✅ **核心标准**：当事人就**将来所要订立合同的主体和标的**达成一致

**预约与本约的区分**：
- 当事人约定在将来一定期限内仍要订立本约合同 → **预约合同**
  - 即使内容已相当完备，甚至比一些本约合同还要全面
- 当事人已履行本属于本约合同的义务（如交付房屋或收受购房款） → **本约合同**
  - 通过履行行为订立了本约合同

**违反预约合同的认定**：
- ✅ 一方要求订立预约合同而对方明确拒绝
- ✅ 一方在磋商订立本约合时时违背诚信原则导致未能订立本约合同
  - 提出的条件明显背离预约合同约定的内容
  - 拒绝讨价还价（未尽力协商）

**违反预约合同的赔偿范围**：
- **基本原则**：不以本约合同的履行利益为标准（否则预约与本约区分无意义）
- **具体标准**：根据交易的成熟度在信赖利益和履行利益之间酌定
  - **交易的成熟度**高 → 接近履行利益
  - **交易的成熟度**低 → 接近信赖利益
- **交易的成熟度**取决于：
  - 预约合同就将来所要订立合同的内容所作约定的完备程度
  - 订立本约合同的条件的成就程度

**实务要点**：
- 如果预约合同已就本约合同的主要内容进行全面约定，且不存在阻碍本约合同订立的情况 → 可**参照本约合同的履行利益**认定赔偿范围

#### 案例2：无权处分与合同效力

**合同效力与合同履行的关系**：
- **无效合同**：不存在请求继续履行
- **有效合同**：也存在不能请求强制履行的情形（《民法典》第580条）
  - **无权处分订立的合同**：有效但不能请求继续履行（法律上不能履行）

**无权处分与善意取得的关系**：
- 无权处分所订合同**有效**
- 标的物已交付或登记 → 买受人**不能自动取得所有权**
- 应适用《民法典》第311条**善意取得制度**
  - 符合善意取得条件 → 取得所有权
  - 不符合善意取得条件 → 物权不发生变动，真正权利人可请求返还

**无权处分规则的体系效应**：
- ✅ 类推适用到其他合同（抵押合同、质押合同等）
- ✅ 转让他人的其他财产权利或设立担保物权 → 参照适用

#### 案例3：以物抵债协议的履行

**以物抵债协议的分类**：
1. **清偿型的以物抵债**（债务履行期限届满后）：
   - 性质：**诺成合同**（非实践合同）
   - 与原债权债务的关系：**新债清偿**
   - 救济：债权人可选择请求履行原债务或以物抵债协议

2. **担保型的以物抵债**（债务履行期限届满前）：
   - 性质：**让与担保合同**
   - 约定"可以对抵债财产拍卖、变卖、折价以实现债权" → **有效**
   - 约定"抵债财产即归债权人所有" → **无效**（但不影响其他部分效力）
   - 已移转财产权利 → 可依据《民法典担保制度司法解释》第68条主张优先受偿

**实务要点**：
- 以物抵债协议 → 不直接引起物权变动
- 人民法院出具的司法确认书或调解书 → 仅确认协议，不直接引起物权变动

### 2.4 操作建议

1. **理解具体制度时**：
   - 必须顾及该制度对其他规则的影响
   - "牵一发而动全身"

2. **处理合同纠纷时**：
   - 检查是否涉及物权变动（善意取得、优先购买权等）
   - 检查是否涉及商事特别法（公司法、证券法等）
   - 检查是否涉及公法规范（审批、备案等）

3. **适用法律时**：
   - 优先适用特别法
   - 特别法无规定 → 适用一般法
   - 避免"张冠李戴"

## 三、辩证思维 (Dialectical Thinking)

### 3.1 核心要义

> "辩证思维是反映和符合客观事物辩证发展过程及其规律性的思维，其特点是从对象的内在矛盾的运动变化中，从其各个方面的相互联系中进行考察，进而从整体上、本质上完整地认识对象。"

**基本要求**：
- ✅ 妥当处理平等保护和倾斜保护的辩证关系
- ✅ 透过现象看本质（穿透式审判）
- ✅ 把握从量变到质变的辩证关系（比例原则）

### 3.2 应用指引

#### 平等保护vs倾斜保护决策树

```
问题：是否应对某一方当事人进行倾斜保护？
│
├─ 情形A：法律地位平等且谈判能力对等
│   └─ ✅ 平等保护（严格贯彻平等原则）
│
├─ 情形B：法律地位平等但缔约地位有差异
│   └─ ⚠️ 倾斜保护（实现实质平等）
│
└─ 情形C：缺乏判断能力导致交易明显不公平
    └─ ⚠️ 倾斜保护（《民法典》第151条）
```

**典型倾斜保护情形**：
- 消费者合同（《消费者权益保护法》+《民法典》格式条款规制）
- 格式条款（《民法典》第496-498条）
- 缺乏判断能力（《民法典合同编通则解释》第11条）

#### 穿透式审判适用边界清单

**适用场景**：
- ✅ 循环贸易（走单不走货）
- ✅ 票据纠纷（清单交易或封包交易）
- ✅ 融资租赁纠纷（只融资不融物）
- ✅ "阴阳合同"（虚假意思表示+隐藏行为）
- ✅ "名实不符"（名为联营实为借贷、名为合作开发实为借贷）

**穿透方法**：
- 适用《民法典》第146条第1款认定虚假意思表示订立的合同**无效**
- 适用《民法典》第146条第2款认定被隐藏合同的效力

**适用边界**：
- ⚠️ 不是武断地脱离当事人约定依照有名合同强行归类
- ✅ 要以当事人约定为依据，探求当事人的真实意思
- ✅ 当存在无法归类于某一有名合同的交易行为时，只要不触犯"违法"或"公序良俗"条款，原则上应尊重当事人的约定

#### 情势变更vs商业风险区分标准表

| 区分维度 | 商业风险 | 情势变更 |
|---------|---------|----------|
| **价格波动程度** | 正常的价格变动 | 政策调整或市场供求关系异常变动导致难以合理预见的涨跌 |
| **可预见性** | 订立合同时可合理预见 | 订立合同时难以合理预见 |
| **履行后果** | 按原定价格履行合同未造成显失公平 | 按原定价格履行合同将带来显失公平的结果 |
| **交易类型** | 大宗商品、股票、期货等风险投资型金融产品 | 其他类型交易 |

**特殊规则**：
- 大宗商品、股票、期货等风险投资型金融产品 → 即使发生异常变动 → **也不认定情势变更**

#### 公序良俗适用的比例原则审查表

**认定合同是否因违背公序良俗无效时**：应坚持**比例原则**

**考量因素**：
1. 当事人的主观动机
2. 交易目的
3. 政府部门的监管强度
4. 一定期限内当事人从事类似交易的频次
5. 行为的社会后果

**程度要求**：
- 给社会公共秩序造成**重大影响** → 认定无效
- 影响显著轻微且不影响国家安全、不违背善良风俗 → **不认定无效**

**裁判文书说理要求**：
- ✅ 充分说理
- ✅ 明确指出合同违背公序良俗的哪个方面
- ✅ 为什么该方面受到损害
- ✅ 为什么损害程度达到"重大"标准

### 3.3 典型实务案例

#### 案例1：格式条款的规制（倾斜保护）

**格式条款的认定**：
- 基本特征：**预先拟定，并在订立合同时未与对方协商**
- "为了重复使用"：仅为拟定目的，**非要求实际被重复使用**

**格式条款订入合同的规则**：
- 提供格式条款一方**履行提示义务或者说明义务**
- 限制范围："免除或者减轻其责任等与对方有**重大利害关系**的条款"
- "与对方有重大利害关系的条款" → 限制在**"异常条款"**
  - 免除或者减轻其责任
  - 排除或者限制对方权利

**电子交易的提示义务**：
- 即使采取设置勾选、弹窗等方式 → **仍应举证证明**已依法履行提示义务或说明义务
- 采取默示勾选方式进行捆绑式销售 → **不认定履行提示义务**

#### 案例2："阴阳合同"的处理（穿透式审判）

**定义**：
- "阳合同"：以虚假意思表示订立用于备案、批准或办理登记
- "阴合同"：以真实意思表示订立用于实际履行

**效力认定**：
- **严格适用**《民法典》第146条
  - "阳合同"（虚假意思表示）→ **无效**
  - "阴合同"（隐藏行为）→ **"依照有关法律规定处理"**
    - 规避的是法律、行政法规的强制性规定 → 依据《民法典》第153条第1款
    - 规避的是法律、行政法规关于合同应当办理批准等手续的规定 → 依据《民法典》第502条第2款

**事实基础**：
- "阴阳合同"均被认定无效或确定不发生效力 → 原则上以"阴合同"作为处理当事人之间权利义务关系的事实基础
  - 防止发生不诚信行为
  - 除非法律另有特别规定

**与合同变更的区分**：
- "阴阳合同"：以虚假意思表示隐藏真实意思表示
- 合同变更：合同有效为前提，当事人协商一致变更内容
  - 一些法律、行政法规（如《招标投标法》第46条）禁止或限制变更 → 该变更应被认定无效

#### 案例3："名实不符"的处理（穿透式审判）

**两种情形**：
1. **合同名称与合同约定的内容不一致**：
   - 涉及合同解释问题
   - 例如："名为联营，实为借贷""名为合作开发，实为借贷"

2. **当事人主张的法律关系与合同约定的权利义务关系不一致**：
   - 涉及当事人以虚假意思表示隐藏真实意思表示
   - 应适用《民法典》第146条

**认定方法**：
- 结合以下事实认定当事人之间的民事法律关系：
  - 缔约背景
  - 交易目的
  - 交易结构
  - 履行行为
  - 当事人是否存在虚构交易标的等事实

**典型案例**：银行采取过桥方式从事票据业务
- 签订的是票据贴现合同
- 但资金流转与票据流转顺序不同于正常票据贴现
  - 出资银行"倒打款"
  - 过桥银行不垫资
- 应严格审查是票据贴现关系还是资金通道服务关系
- 参考《民商审判会议纪要》第103条、第104条处理

### 3.4 操作建议

1. **处理格式条款纠纷时**：
   - 审查是否属于格式条款（预先拟定+未协商）
   - 审查是否履行提示义务或说明义务（异常条款）
   - 电子交易中经营者需举证证明已履行提示义务

2. **处理"阴阳合同"纠纷时**：
   - 严格适用《民法典》第146条（虚伪表示+隐藏行为）
   - 原则上以"阴合同"作为事实基础

3. **处理"名实不符"纠纷时**：
   - 综合考察缔约背景、交易目的、交易结构、履行行为等
   - 探求真实意思，还原事物本来面目

4. **认定违背公序良俗时**：
   - 坚持比例原则
   - 综合考量5个因素（主观动机、交易目的、监管强度、交易频次、社会后果）
   - 充分说理

## 四、程序思维 (Procedural Thinking)

### 4.1 核心要义

> "要坚持案结事了政通人和理念。把实质性化解矛盾、解决问题作为司法审判的目标、导向，以'如我在诉'的意识，在每一个审判环节都把服判息诉的功课做到极致。"

**基本要求**：
- ✅ 实质性化解纠纷（避免程序空转）
- ✅ 诉源治理或执源治理（在审判阶段兼顾执行）
- ✅ 构建多元纠纷解决机制
- ✅ 合理分配证明责任

### 4.2 应用指引

#### 释明权行使规则清单

**何时行使释明权**：
- 当事人主张的法律关系性质或者民事行为效力与人民法院根据案件事实作出的认定不一致时

**释明内容**：
- 告知当事人变更诉讼请求

**典型情形**：
- 基于合同有给付行为的原告请求确认合同无效，但未提出返还原物或折价补偿、赔偿损失等请求
  - **应向其释明**：一并提出相应诉讼请求
- 原告请求确认合同无效并要求被告返还原物或赔偿损失，被告基于合同也有给付行为
  - **应向被告释明**：也可以提出返还请求

**特殊规定**（《民事诉讼证据规定》第53条）：
- 即使当事人没有根据法庭辩论的情况申请变更诉讼请求
- 也应在判决书中**直接对当事人之间的法律关系性质及民事行为效力作出认定**
- 可以根据案件的具体情况重新指定举证期限

#### 合并审理适用情形表

| 纠纷类型 | 合并审理条件 | 法律依据 |
|---------|-------------|----------|
| **代位权诉讼** | 两个以上债权人以债务人的同一相对人为被告 | 《民法典合同编通则解释》第37条 |
| **代位权诉讼+起诉债务人** | 债权人起诉债务人后又对债务人的相对人提起代位权诉讼，且属于同一人民法院管辖 | 《民法典合同编通则解释》第38条 |
| **撤销权诉讼** | 债权人请求一并审理其与债务人之间的债权债务关系，且属于该人民法院管辖 | 《民法典合同编通则解释》第44条 |

**合并审理的实务要点**：
- 债务人对相对人享有的债权不足以清偿其对两个以上债权人负担的债务
  - 人民法院可以按照债权人享有的债权比例确定相对人的履行份额

#### 举证责任动态调整机制

**违约金调整的举证责任分配**：
1. **客观证明责任**（立法者在实体法中分配）：
   - 主张违约金过高的当事人 → 承担举证证明责任
   - 主张违约金低于实际损失的当事人 → 承担举证证明责任

2. **主观证明责任**（法官在具体案件中分配）：
   - 违约方举出**初步证据**使法官产生**合理怀疑**
   - 法官应将主观证明责任**移转给守约方**
   - 守约方需举证推翻这一合理怀疑
   - 如果守约方不能举证推翻 → 法官即可产生违约金过高的内心确信

**理由**：
- 损失发生在守约方
- 让违约方举证证明守约方的损失且达到高度可能性 → 实属难为

#### 裁判文书执行力预判清单

**同时履行抗辩权的判决表述**：
- 判项中明确载明："原告申请强制执行的，人民法院应当在原告履行自己的债务后对被告采取执行行为"
- **性质**：并非将原告履行自己的义务作为申请执行的条件
- **而是**：作为人民法院采取执行行为的条件

**先履行抗辩权的判决表述**：
- 应驳回原告的诉讼请求
- 但应明确原告在履行债务后可以另行提起诉讼

**撤销权诉讼中的返还请求**：
- 债权人在撤销权诉讼中可以同时请求相对人向债务人承担返还财产、折价补偿、履行到期债务等法律后果
- 债权人依据生效法律文书申请强制执行
- 人民法院可以就债务人对相对人享有的权利采取强制执行措施以实现债权人的债权

### 4.3 典型实务案例

#### 案例1：可得利益损失的认定（事实认定）

**《民法典》第584条**：
- 守约方可主张的损失：
  - 因违约所遭受的损失
  - 合同履行后可以获得的利益（**可得利益**）
- 限制：可预见性规则

**可得利益的计算**：
- **扣除合理成本**：在确定可得利益时，应扣除守约方为订立、履行合同支出的费用等合理成本
- **不得重复计算**：不能将为取得利益而应当支出的成本作为因违约而遭受的损失重复计算

**可预见性规则的适用**：
- 综合考量因素：
  - 合同主体
  - 合同内容
  - 交易类型
  - 交易习惯
  - 磋商过程

**证明标准**：
- 不宜要求过高
- 在守约方不能举证证明其主张的损失具有高度可能性时 → 人民法院应根据案件具体情况**酌定**违约方的损失赔偿额

**可得利益的计算方法**：
1. **替代交易法**：
   - 守约方依法行使合同解除权并实施了替代交易
   - 按照替代交易价格与合同价格的差额确定
   - **例外**：替代交易价格明显偏离市场价格 → 按市场价格与合同价格的差额确定

2. **市场价格法**：
   - 守约方依法行使合同解除权但未实施替代交易
   - 按照违约行为发生后合理期间内合同履行地的市场价格与合同价格的差额确定

3. **持续性定期合同的特殊规则**：
   - 根据合同主体、交易类型、市场价格变化、剩余履行期限等因素
   - 确定非违约方寻找替代交易的合理期限
   - 按照该期限对应的价款、租金等扣除非违约方应当支付的相应履约成本确定

4. **酌定法**：
   - 根据上述规则仍难以确定时
   - 综合考虑以下因素：
     - 违约方因违约获得的利益
     - 违约方的过错程度
     - 其他违约情节
   - 遵循公平原则、诚信原则确定

#### 案例2：违约金调整的举证责任（证明责任分配）

**违约金的性质和功能**：
- **损失预定**
- **分配证明责任**

**举证责任的动态移转**：
- 违约方举出初步证据使法官产生**合理怀疑**
- 法官应将主观证明责任移转给守约方
- 守约方举证推翻这一合理怀疑
- 如果守约方不能举证推翻 → 法官即可产生违约金过高的内心确信

**"合理怀疑"的判断标准**：
- 需要法官根据案件具体情况自由心证
- 典型情形：违约金超过损失30%（一般认定过高）

#### 案例3：代位权诉讼与仲裁的关系

**问题**：债务人与其相对人之间存在仲裁协议，债权人如何行使代位权？

**处理方案**：
- 一方面：尊重仲裁协议
- 另一方面：防止当事人利用仲裁协议架空代位权诉讼

**具体规则**：
- 即使债务人与其相对人之间存在仲裁协议 → 债权人也可对债务人的相对人提起代位权诉讼
- **例外**：债务人或者其相对人在首次开庭前已经申请仲裁 → 代位权诉讼应当中止，等待仲裁裁决的结果再恢复代位权诉讼
- 如果在首次开庭前未申请仲裁 → 债务人或者其相对人再以存在仲裁协议为由主张人民法院对双方之间的纠纷没有管辖权 → 人民法院不予支持

### 4.4 操作建议

1. **行使释明权时**：
   - 当事人主张的法律关系性质或效力与法院认定不一致时 → 应释明变更诉讼请求
   - 基于合同有给付行为的原告请求确认合同无效 → 应释明一并提出返还或赔偿请求

2. **合并审理时**：
   - 灵活运用合并审理制度
   - 避免程序空转，实质性化解纠纷

3. **撰写裁判文书时**：
   - 预判执行力问题
   - 同时履行抗辩权：在判项中明确载明执行条件
   - 先履行抗辩权：驳回原告诉讼请求，但明确原告可另行起诉

4. **分配举证责任时**：
   - 违约金调整：动态移转主观证明责任
   - 可得利益：酌定损失赔偿额（证明标准不宜要求过高）

### 5.2 典型案例：强制规定但书的适用（融合四维思维）

**《民法典合同编通则解释》第16条第1款**：违反强制性规定不导致合同无效的5种情形

#### 历史思维的应用
- **制度演进**：从"违反法律和政策即无效"→"违反法律行政法规强制性规定"→"效力性强制规定"→"强制性规定但书"
- **守正**：继续限制合同无效的范围
- **创新**：直接针对但书作出解释（不再采用"效力性强制规定"表述）

#### 体系思维的应用
- **体系定位**：但书是《民法典》第153条第1款的一部分，需与公序良俗（第2款）体系解释
- **体系逆反纠正**：限制民事权利的"不得""必须""应当" → 不适用第153条第1款（应适用无权处分、越权代表等规则）

#### 辩证思维的应用
- **比例原则**：情形1"影响显著轻微"（比例原则在民法上的具体运用）
  - "举重以明轻"：《刑法》第13条"情节显著轻微危害不大的，不认为是犯罪" → 民法更应如此
- **利益衡量**：情形2"旨在维护国家利益"（仅涉及税收、土地出让金等）
  - 认定有效不影响规范目的实现

#### 程序思维的应用
- **司法建议和移送机制**：在上述例外情形下，当事人从违法行为获得利益
  - 向有关行政管理部门提出司法建议
  - 涉嫌犯罪的 → 将案件线索移送刑事侦查机关
- **实质性化解纠纷**：民事、行政、刑事审判互为补充，形成合力

### 5.3 操作建议

1. **分析案件时**：
   - 先运用**历史思维**：了解制度发展脉络
   - 再运用**体系思维**：避免"只见树木不见森林"
   - 然后运用**辩证思维**：透过现象看本质
   - 最后运用**程序思维**：实质性化解纠纷

2. **撰写裁判文书时**：
   - 历史思维：说明制度演进和司法政策延续性
   - 体系思维：举重以明轻，避免体系逆反
   - 辩证思维：充分说理，体现比例原则
   - 程序思维：预判执行力，避免程序空转

3. **提升裁判质量时**：
   - 系统学习和运用四维思维方法
   - "授人以鱼不如授人以渔"
   - 通过思维方法的系统运用，确保《民法典》得到正确实施
