### 四、合同起草要点

#### 4.1 当事人信息
- 自然人：姓名、身份证号、住址、联系电话
- 法人：名称、统一社会信用代码、住所、法定代表人、联系电话

#### 4.2 标的条款
- 标的物的名称、数量、质量、规格、型号
- 标的物的权属状况

#### 4.3 价款条款
- 金额、币种
- 支付方式（现金、转账、支票等）
- 支付时间
- 发票条款

#### 4.4 履行条款
- 履行时间
- 履行地点
- 履行方式
- 履行顺序

#### 4.5 违约责任
- 违约情形
- 违约责任承担方式（继续履行、赔偿损失、支付违约金）
- 违约金数额或计算方式

#### 4.6 争议解决
- 协商、调解、仲裁、诉讼
- 仲裁条款（明确仲裁机构）
- 管辖法院条款

#### 4.7 其他条款
- 不可抗力
- 保密条款
- 知识产权条款
- 通知条款
- 生效条款

### 五、法律文书模板要点

#### 5.1 起诉状要点

**结构**：
- 原告信息
- 被告信息
- 诉讼请求（明确、具体、可执行）
- 事实和理由（事实+法律依据）
- 证据和证据来源
- 致送法院
- 落款

**注意事项**：
- 诉讼请求要明确、具体、可执行
- 事实和理由要分开写
- 法律依据要引用准确

#### 5.2 答辩状要点

**结构**：
- 答辩人信息
- 被答辩人信息
- 答辩请求（驳回起诉、驳回诉讼请求等）
- 事实和理由（针对性反驳）
- 证据和证据来源
- 致送法院
- 落款

**注意事项**：
- 针对原告的诉讼请求逐一反驳
- 事实和理由要针对性强
- 引用法律依据要准确

#### 5.3 律师函要点

**结构**：
- 标题（律师函）
- 致送对象
- 事实陈述
- 法律分析
- 律师意见（要求、警告）
- 法律后果提示
- 落款（律师事务所、律师、日期）

**注意事项**：
- 事实陈述要客观
- 法律分析要准确
- 要求要明确、具体
- 法律后果提示要充分

### 六、合同审查清单

#### 6.1 合同效力审查
- [ ] 合同主体是否适格
- [ ] 合同形式是否合法
- [ ] 合同内容是否违反强制性规定
- [ ] 合同内容是否违背公序良俗

#### 6.2 权利义务审查
- [ ] 权利义务是否明确
- [ ] 权利义务是否对等
- [ ] 是否存在显失公平的条款
- [ ] 是否存在格式条款的不公平条款

#### 6.3 违约责任审查
- [ ] 违约情形是否明确
- [ ] 违约责任承担方式是否合理
- [ ] 违约金是否过高或过低
- [ ] 定金是否超过合同额的20%

#### 6.4 解除条款审查
- [ ] 约定解除条件是否明确
- [ ] 法定解除条件是否完备
- [ ] 解除程序是否明确
- [ ] 解除后果是否明确

#### 6.5 争议解决审查
- [ ] 管辖条款是否有效
- [ ] 仲裁条款是否明确
- [ ] 是否选择或裁或审

#### 6.6 合规性审查
- [ ] 是否符合行业监管规定
- [ ] 是否符合反腐败、反垄断、反洗钱要求
- [ ] 是否符合数据安全、个人信息保护要求
- [ ] 是否符合外汇、税收规定
