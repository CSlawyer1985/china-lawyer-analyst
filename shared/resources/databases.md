# 法律数据库指引 (Legal Databases Guide)

---

## 模块说明

本模块是 china-lawyer-analyst v2.0 的共享资源模块，提供各类法律数据库的使用指引。

**适用领域**：所有领域

**依赖模块**：shared/methods/legal-research.md

**触发场景**：需要查找法律资源时

---

## 内容

### 一、付费数据库（专业）

#### 1.1 威科先行

**网址**：https://law.wkinfo.com.cn/

**覆盖范围**：
- 法律、法规、司法解释
- 案例（最高法、各高院、中院、基层法院）
- 实务文章（律师、法官、学者撰写）
- 合同范本（各类合同模板）

**优势**：
- 内容全面：覆盖中国法律体系的主要组成部分
- 更新及时：实时更新最新法律法规、司法解释
- 检索功能强大：支持多维度组合检索
- 实务导向：提供大量实务文章和合同范本

**适用场景**：
- 综合法律检索
- 案例检索
- 实务研究
- 合同起草

#### 1.2 北大法宝

**网址**：https://www.pkulaw.com/

**覆盖范围**：
- 法律、法规、司法解释
- 案例（指导案例、公报案例、典型案例）
- 法学期刊（学术文章）
- 法律法规英文译本

**优势**：
- 权威性强：北京大学法学院出品
- 学术资源丰富：收录大量法学期刊文章
- 英译本完善：提供法律法规的英文翻译
- 检索精度高：支持法条关联检索

**适用场景**：
- 学术研究
- 法律条文检索
- 案例检索
- 英文法律研究

#### 1.3 法信

**网址**：https://www.faxin.cn/

**覆盖范围**：
- 最高法院案例
- 裁判规则（提炼自案例）
- 实务指南（最高法院发布）
- 法律法规

**优势**：
- 案例资源丰富：收录大量最高法院案例
- 裁判规则提炼：将案例中的裁判规则进行提炼
- 实务性强：提供大量实务操作指南
- 最高法院背景：由最高人民法院主管

**适用场景**：
- 案例检索
- 裁判规则研究
- 实务指南查询
- 最高法院文件检索

#### 1.4 无讼案例

**网址**：https://www.itslaw.com/

**覆盖范围**：
- 裁判文书
- 实务观点（律师、法官观点）
- 律师办案工具
- 法律法规

**优势**：
- 实务导向：面向律师实务
- 律师视角：提供律师办案经验和观点
- 工具丰富：提供律师办案所需的各类工具
- 用户互动：支持律师之间的交流

**适用场景**：
- 律师办案
- 实务观点查询
- 裁判文书检索
- 律师交流

### 二、免费资源

#### 2.1 中国裁判文书网

**网址**：https://wenshu.court.gov.cn/

**覆盖范围**：
- 各级法院裁判文书

**优势**：
- 官方权威：最高人民法院主办
- 免费公开：所有文书免费查阅
- 覆盖面广：覆盖各级法院

**适用场景**：
- 裁判文书检索
- 类案检索
- 诉讼风险评估

#### 2.2 国家法律法规数据库

**网址**：https://flk.npc.gov.cn/

**覆盖范围**：
- 法律
- 行政法规
- 部门规章

**优势**：
- 官方权威：全国人大常委会主办
- 免费公开：所有法律法规免费查阅
- 更新及时：实时更新最新法律法规

**适用场景**：
- 法律法规检索
- 条文验证
- 最新法律法规查询

#### 2.3 中国法院网

**网址**：https://www.chinacourt.org/

**覆盖范围**：
- 法律法规
- 最高法文件
- 典型案例
- 法院新闻

**优势**：
- 官方权威：最高人民法院主管
- 信息全面：覆盖法律法规、案例、新闻
- 更新及时：实时更新最新动态

**适用场景**：
- 法律法规检索
- 最高法文件检索
- 典型案例查询
- 法院动态了解

#### 2.4 各地法院官网

**示例**：
- 北京法院网
- 上海法院网
- 浙江法院网

**覆盖范围**：
- 典型案例
- 审判白皮书
- 地方司法文件
- 法院动态

**优势**：
- 地方特色：提供地方典型案例和司法文件
- 实务指导：提供审判白皮书等实务指导
- 官方权威：各地法院主办

**适用场景**：
- 地方典型案例查询
- 地方司法文件查询
- 审判白皮书查阅

### 三、数据库选择指南

#### 3.1 按检索目标选择

| 检索目标 | 推荐数据库 | 备选数据库 |
|---------|-----------|-----------|
| 法律条文 | 威科先行、北大法宝 | 国家法律法规数据库 |
| 司法案例 | 法信、无讼案例 | 中国裁判文书网 |
| 实务文章 | 威科先行 | 无讼案例 |
| 合同范本 | 威科先行 | 无讼案例 |
| 学术文章 | 北大法宝 | 中国知网 |
| 英文法律 | 北大法宝 | 威科先行 |

#### 3.2 按专业领域选择

| 专业领域 | 推荐数据库 | 特点 |
|---------|-----------|------|
| 公司法 | 威科先行、北大法宝 | 案例丰富、实务文章多 |
| 建筑工程 | 法信、威科先行 | 最高法院案例多、实务指南全 |
| 知识产权 | 北大法宝、无讼案例 | 学术性强、实务观点多 |
| 劳动法 | 威科先行、法信 | 地方司法文件多、裁判规则全 |
| 刑法 | 北大法宝、法信 | 学术资源丰富、案例权威 |

### 五、常见误区

**误区1：只使用免费数据库**
- 正确做法：综合使用付费和免费数据库，提高检索质量

**误区2：忽视数据库的权威性**
- 正确做法：优先使用官方权威数据库（如国家法律法规数据库、中国裁判文书网）

**误区3：不掌握高级检索技巧**
- 正确做法：学习并掌握各数据库的高级检索技巧，提高检索效率

**误区4：忽视数据库的更新频率**
- 正确做法：关注数据库的更新频率，优先使用更新及时的数据库
