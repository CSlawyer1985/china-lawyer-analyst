### 二、检索策略

#### 2.1 六步检索法

**步骤 1：明确法律问题**
- 案件的核心法律问题是什么？
- 涉及哪些法律领域（合同、侵权、公司、劳动等）？
- 需要检索什么类型的资料（法律、案例、实务文章）？

**步骤 2：确定检索关键词**
- 使用法言法语（法律专业术语）
- 示例：
  - 合同纠纷："违约责任""合同解除""损害赔偿"
  - 侵权纠纷："安全保障义务""产品责任""过错推定"
  - 公司纠纷："股权转让""股东代表诉讼""公司法人格否认"

**步骤 3：选择合适的数据库**
- 综合检索：威科先行、北大法宝
- 案例检索：法信、无讼案例、中国裁判文书网
- 学术检索：中国知网（CNKI）、北大法宝法学期刊

**步骤 4：组合检索**
- **关键词 + 法律层级**：如"违约责任 + 民法典"
- **关键词 + 时间范围**：如"建设工程施工合同 + 2020-2024"
- **关键词 + 法院层级**：如"指导案例 + 最高法"
- **关键词 + 案由**：如"买卖合同纠纷 + 质量瑕疵"

**步骤 5：筛选相关结果**
- 按相关性排序
- 按时间排序（最新的优先）
- 按效力筛选（现行有效）

**步骤 6：验证时效性**
- 法律是否现行有效？
- 是否有新的司法解释？
- 是否有新的指导案例？

#### 2.2 检索技巧

**布尔检索**：
- AND（与）：违约 AND 侵权
- OR（或）：合同 OR 协议
- NOT（非）：侵权 NOT 刑事
- ""（精确匹配）："违约责任"
- *（通配符）：违约*（匹配违约、违约金、违约责任）

**高级检索**：
- 限定法律层级：法律类型=法律
- 限定时间范围：发布日期 BETWEEN 2020-01-01 AND 2024-12-31
- 限定法院层级：法院层级=最高人民法院
- 限定案由：案由=买卖合同纠纷

**类案检索**：
- 按案由检索
- 按法院层级检索
- 按时间范围检索
- 按关键词检索（争议焦点、裁判要点）

### 三、检索结果的应用

#### 3.1 法律依据
- 引用现行有效的法律、行政法规、司法解释
- 注意法律的效力等级（上位法优于下位法）
- 注意法律的时效性（新法优于旧法）

#### 3.2 案例支持
- 参照指导案例的裁判要点
- 参照公报案例、典型案例的裁判规则
- 运用区别技术区分事实差异

#### 3.3 实务观点
- 参考实务文章的观点
- 参考学者、律师、法官的观点
- 结合具体案情分析

---

### 四、检索质量控制

#### 4.1 准确性验证
- 核对条文号、内容是否准确
- 确认法律、司法解释是否现行有效
- 确认案例是否被撤销或改判

#### 4.2 完整性验证
- 是否遗漏了重要法律依据
- 是否遗漏了重要类案
- 是否考虑了相反观点

#### 4.3 时效性验证
- 查看法律的施行时间、修正时间
- 查看司法解释的发布时间、是否失效
- 查看案例的裁判时间、是否被后续案例推翻

---

### 五、常见误区

**误区1：只检索法律条文，忽视司法解释**
- 正确做法：同时检索法律、司法解释、案例

**误区2：忽视法律的时效性**
- 正确做法：验证法律是否现行有效、是否有最新修正

**误区3：过度依赖免费数据库**
- 正确做法：综合使用付费和免费数据库，提高检索质量

**误区4：关键词使用不当**
- 正确做法：使用法言法语，避免口语化表达

### 六、民法典合同编通则司法解释（2023）检索指引

**重要提示**: 本章节为 v2.1.0 新增内容，整合2023年最新司法解释的检索要点。

#### 6.1 官方文本

**来源**:
- 最高人民法院官网
- 中国裁判文书网
- 国家法律法规数据库

**引用格式**:
- 【司法解释】《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》第X条

#### 6.2 理解与适用

**书籍**:
- 最高人民法院民事审判第二庭、研究室编著：《最高人民法院民法典合同编通则司法解释理解与适用》，人民法院出版社2023年版

**特点**:
- 含有条文释义、起草背景、实务应用
- 含有典型案例分析
- 含有不同观点的辨析

#### 6.3 检索技巧

**关键词检索**:
- 条文序号（如"第十六条"）
- 核心概念（如"违反强制性规定"、"预约合同"、"越权代表"）
- 结合具体案情（如"商品房买卖"、"担保合同"）

**类案检索**:
- 威科先行、北大法宝、法信
- 关键词：相关条文 + 案由（如"第十六条" + "合同纠纷"）

#### 6.4 适用顺序

**合同纠纷分析检索顺序**:
1. 民法典合同编条文
2. **民法典合同编通则司法解释（2023）** ⭐⭐⭐⭐⭐
3. 民法典担保制度司法解释（如涉及担保）
4. 其他相关司法解释
5. 指导性案例、公报案例、典型案例

**注意事项**:
- 《合同法司法解释（一）》《合同法司法解释（二）》已废止
- 该司法解释施行后尚未终审的案件适用
- 施行前已经终审的案件不适用

#### 7.2 校验流程

**步骤1: 提取法律引用**
- 自动提取法律条文：《中华人民共和国XXX法》第XX条
- 自动提取司法解释：法释〔20XX〕XX号
- 自动提取案例：最高人民法院第XX号指导性案例
- 自动提取时间点：案件事实发生时间

**步骤2: 批量检查更新**
- 查询法律元数据库（`interpretations/metadata.json`）
- 检查是否有最新修正版
- 检查是否有替代性新法
- 检查是否已废止或失效

**步骤3: 新旧法适用判断**
- **原则1: 法不溯及既往（实体法）**
  - 事实发生在新法实施前 → 适用旧法
  - 事实发生在新法实施后 → 适用新法
- **原则2: 程序法从新**
  - 程序问题一律适用新法
- **原则3: 过渡期特殊规则**
  - 检查是否有过渡条款

**步骤4: 官方数据库验证**
- 优先查询：国家法律法规数据库（https://flk.npc.gov.cn）⭐⭐⭐⭐⭐
- 优先查询：最高法院官网（http://www.court.gov.cn）⭐⭐⭐⭐⭐
- 补充查询：Gety本地文档库、Unifuncs网页搜索

**步骤5: 生成校验报告**
- 检查概况：涉及法规数量、发现问题数量
- 详细结果：每个法规的校验结果
- 新旧法适用判断：适用理由、判断依据
- 校验结论：总体结论和建议

**步骤6: 应用修正（如需要）**
- 标记问题法规：~~《公司法》(2018修正)~~
- 检索最新法规全文
- 更新法律引用和条文内容
- 重新IRAC分析
- 生成修正后意见

#### 7.4 新旧法适用原则

**原则1: 法不溯及既往（实体法）**

**定义**:
- 法律、行政法规、地方性法规、自治条例和单行条例、规章不溯及既往
- 但为了更好地保护公民、法人和其他组织的权利和利益而作的特别规定除外

**适用**:
- **实体法**（如民法典、公司法）：适用旧法
  - 事实发生在新法实施前 → 适用旧法
  - 事实发生在新法实施后 → 适用新法
- **示例**:
  ```
  案例：2023年6月发生的股权转让纠纷
  分析：事实发生在2024年7月1日（新公司法实施）前
  结论：适用《公司法》（2018年修正），不适用（2023年修订）
  ```

**原则2: 程序法从新**

**定义**:
- 程序问题（诉讼程序、管辖等）适用新法

**适用**:
- **程序法**（如民事诉讼法）：适用新法
  - 无论事实发生在何时，程序问题一律适用新法
- **示例**:
  ```
  案例：2023年6月发生的纠纷，2026年提起诉讼
  分析：实体问题适用旧法，但程序问题（起诉、管辖）适用新法
  结论：实体判断适用《公司法》（2018年修正）
        程序问题适用《民事诉讼法》（2021年修正）
  ```

**原则3: 过渡期特殊规则**

**定义**:
- 某些法律设有过渡期，过渡期内适用特殊规则

**适用**:
- 检查是否有过渡期规定
- 适用特殊过渡规则

**示例**:
```
案例：2024年8月的出资纠纷
新公司法：2024年7月1日实施，3年过渡期（至2027年6月30日）
分析：在过渡期内，适用过渡期特殊规则
结论：适用新公司法第228条过渡期规定
```

**常见法律的实施时间**:

| 法律 | 旧法实施时间 | 新法实施时间 | 过渡期 |
|------|-------------|-------------|--------|
| 公司法 | 2013-01-01 | 2024-07-01 | 2024-07-01至2027-06-30 |
| 民法典 | - | 2021-01-01 | 无 |
| 民事诉讼法 | 2013-01-01 | 2022-01-01 | 无 |
| 劳动合同法 | 2008-01-01 | 2013-07-01 | 无 |

**示例2: 自动判断新旧法适用**

```

系统行为：
→ Step 1: 提取法律引用、事实时间
→ Step 2: 检查更新
  发现2023年修订版，2024-07-01实施
```
