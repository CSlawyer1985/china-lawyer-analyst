### 三、典型合同类型

#### 1. 买卖合同 (第595-640条)
- 标的物所有权转移
- 标的物风险转移
- 质量瑕疵担保
- 检验期限

#### 2. 借款合同 (第667-680条)
- 利率限制(不得违反国家有关规定)
- 自然人借款合同的实践性

#### 3. 租赁合同 (第703-734条)
- 租赁期限(不得超过20年)
- 转租规则
- 买卖不破租赁

#### 4. 承揽合同 (第770-787条)
- 承揽人的工作义务
- 定作人的协助义务
- 承揽人的留置权

### 四、特殊合同规则

#### 1. 建设工程合同 (第788-808条) ★ 重点关注

**合同形式**:
- 应当采用书面形式

**发包人权利义务**:
- 发包人可以随时解除合同(第788条)
- 发包人未按照约定支付价款的,承包人可以催告(第807条)
- 发包人提供材料、技术资料、协作义务

**承包人权利义务**:
- 承包人可以将部分工作交由第三人完成(第791条)
- 承包人对建设工程质量负责
- 承包人建设工程价款优先受偿权(第807条)★

**建设工程价款优先受偿权**:
- 优先于抵押权和其他债权
- 期限:自发包人应当给付建设工程价款之日起算18个月
- 范围:承包人为建设工程应当支付的工作人员报酬、材料款等实际支出的费用

#### 2. 技术合同 (第843-876条)
- 技术开发合同
- 技术转让合同
- 技术咨询合同
- 技术服务合同
- 职务技术成果的归属

### 五、合同纠纷的举证责任

#### 1. 合同成立纠纷
- 主张合同关系成立的一方,承担举证责任(民诉法解释第91条)

#### 2. 合同效力纠纷
- 主张合同无效的一方,承担举证责任(证明违反法律、行政法规的强制性规定或违背公序良俗)

#### 3. 违约责任纠纷
- 主张违约的一方,承担举证责任(证明对方未履行或不符合约定)
- 主张不可抗力免责的一方,承担举证责任(证明不可抗力事件及因果关系)

### 六、常见合同纠纷类型

#### 1. 软件开发合同纠纷

**争议焦点**:
- 延迟交付是否构成违约?
- 软件是否存在重大缺陷?
- 是否可以拒绝付款?
- 损害赔偿如何计算?

**法律适用**:
- 民法典第577条(违约责任)
- 民法典第526条(先履行抗辩权)
- 民法典第582条(补救措施)

**裁判要点**:
- 延迟交付是否构成重大违约,需考虑时间长短、是否造成损失、是否影响合同目的实现
- 软件缺陷是否构成重大违约,需根据技术鉴定确定
- 甲方有权行使先履行抗辩权,但拒付范围应当与违约程度相适应

#### 2. 买卖合同纠纷

**质量瑕疵纠纷**:
- 买受人应当在约定或合理期限内检验(第620-623条)
- 标的物数量、质量不符合约定的,买受人可以请求补救(第582条)

**价款支付纠纷**:
- 买受人应当按照约定的时间、数额支付价款(第626条)
- 出卖人多交标的物的,买受人可以拒收(第604条)

#### 3. 租赁合同纠纷

**租金支付纠纷**:
- 承租人应当按照约定的期限支付租金(第721条)
- 承租人无正当理由未支付租金,出租人可以请求支付(第722条)

**租赁物维修纠纷**:
- 出租人应当履行租赁物的维修义务(第712条)
- 承租人未尽妥善保管义务的,应当承担赔偿责任(第714条)

### 七、合同审查要点清单

#### 合同效力审查
- [ ] 合同主体是否适格(民事行为能力、经营范围)
- [ ] 合同形式是否合法(书面、口头、其他)
- [ ] 合同内容是否违反法律、行政法规的强制性规定
- [ ] 合同内容是否违背公序良俗

#### 权利义务审查
- [ ] 权利义务是否明确
- [ ] 权利义务是否对等
- [ ] 是否存在显失公平的条款
- [ ] 是否存在格式条款的不公平条款

#### 违约责任审查
- [ ] 违约情形是否明确
- [ ] 违约责任承担方式是否合理
- [ ] 违约金是否过高或过低(通常不超过损失的30%)
- [ ] 定金是否超过合同额的20%

#### 解除条款审查
- [ ] 约定解除条件是否明确
- [ ] 法定解除条件是否完备
- [ ] 解除程序是否明确
- [ ] 解除后果是否明确

#### 争议解决审查
- [ ] 管辖条款是否有效
- [ ] 仲裁条款是否明确(仲裁机构、仲裁事项、仲裁地点)
- [ ] 是否选择或裁或审(仲裁或诉讼只能选其一)

### 十、实务技巧

#### 合同起草技巧
1. **明确标的物**: 详细描述标的物的名称、规格、型号、质量标准
2. **明确履行期限**: 约定明确的履行时间,避免"尽快""合理时间"等模糊表述
3. **明确违约责任**: 约定具体的违约情形和责任承担方式
4. **约定争议解决**: 选择明确的管辖法院或仲裁机构

#### 合同审查技巧
1. **审查主体资格**: 核对营业执照、法定代表人身份证明、授权委托书
2. **审查标的物**: 确认标的物是否存在、是否合法
3. **审查权利义务**: 确保权利义务对等,避免显失公平
4. **审查违约责任**: 确认违约金合理,定金不超过合同额的20%

#### 合同纠纷处理技巧
1. **及时主张权利**: 注意诉讼时效(一般为3年)
2. **保全证据**: 保存合同原件、履行记录、沟通记录
3. **评估损失**: 收集损失证据,计算实际损失和可得利益损失
4. **考虑和解**: 评估诉讼成本和风险,考虑调解或和解
