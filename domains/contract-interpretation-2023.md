## 二、十大核心升级点

### 升级点1：合同效力认定的新规则 ⭐⭐⭐⭐⭐

#### 1.1 违反强制性规定不导致合同无效的5种情形（第16条）

**核心原则**: 价值判断和利益衡量

**5种情形**:

| 序号 | 情形 | 核心要点 | 示例 |
|------|------|----------|------|
| 1 | 影响显著轻微 | 比例原则，情节轻微不认定无效 | 小额违规行为 |
| 2 | 旨在维护国家利益 | 认定有效不影响规范目的 | 未支付土地出让金的土地使用权转让 |
| 3 | 要求一方加强风控 | 对方无审查义务 | 首套住房贷款利率限制 |
| 4 | 订立后可补正但不补正 | 违背诚信原则 | 开发商已具备预售条件但不申请 |
| 5 | 其他法律规定的情形 | 兜底条款 | 民法典第706条租赁合同备案 |

**判断逻辑**:
```
违反强制性规定
    ↓
是否导致合同无效？
    ↓
考察因素：
① 规范目的
② 违反程度
③ 是否影响公共利益
④ 是否有公法责任足以实现立法目的
    ↓
结论：
公法责任足以实现立法目的 → 合同有效
仅有行政/刑事责任，不影响合同效力
```

**示例**:
> 案情: 开发商未取得预售许可证明即签订商品房买卖合同，房价上涨后开发商以合同无效为由拒绝履行。

> 分析:
> - 根据《商品房买卖合同司法解释》第2条，起诉前未取得预售许可证明 → 合同无效
> - 但如果起诉前已取得预售许可证明 → 合同有效
> - 如果开发商已具备申请条件但违背诚信原则不申请 → 违背诚信原则，不应支持无效主张

#### 1.2 公序良俗认定的具体标准（第17条）

**违背公序良俗的3种情形**:

1. **影响国家安全**
   - 政治安全
   - 经济安全
   - 军事安全

2. **影响社会公共秩序**
   - 社会稳定
   - 公平竞争秩序
   - 社会公共利益

3. **违背善良风俗**
   - 背离社会公德
   - 家庭伦理
   - 有损人格尊严

**认定因素**（第17条第2款）:
- 当事人的主观动机和交易目的
- 政府部门的监管强度
- 一定期限内类似交易的频次
- 行为的社会后果
- 以社会主义核心价值观为导向

**例外情形**: 确因生活需要，未给社会公共秩序造成重大影响

#### 1.3 权限限制性规定与违反强制性规定的区分（第18条）

**核心规则**:

如果强制性规定旨在限制或赋予民事权利，行为人违反该规定将构成：
- 无权处分
- 无权代理
- 越权代表
- 或者导致合同相对人、第三人获得撤销权、解除权等民事权利

**适用规则**: 依据关于违反该规定的民事法律后果认定合同效力，**不适用民法典第153条第1款**

**示例**:
- 《物权法》第191条"未经抵押权人同意，不得转让抵押财产" → 不导致合同无效，适用无权处分规则
- 《公司法》第16条公司对外担保 → 不导致合同无效，适用越权代表规则

### 升级点2：预约合同的认定与救济 ⭐⭐⭐⭐⭐

#### 2.1 预约合同的认定（第6条）

**成立要件**:

1. **形式要件**:
   - 认购书、订购书、预订书等
   - 或交付立约定金

2. **内容要件**:
   - 能够确定将来所要订立合同的**主体**
   - 能够确定将来所要订立合同的**标的**

3. **意思表示**:
   - 受意思表示约束
   - 未明确约定"不受约束"

**不成立预约合同的情形**:
- 仅表达交易意向（意向书、备忘录）
- 未约定在将来一定期限内订立合同
- 难以确定将来所要订立合同的主体、标的等内容

**预约合同与本约合同的区分（第6条第3款）**:

同时满足以下条件 → **认定本约合同成立**:
- 已就合同标的、数量、价款或者报酬等主要内容达成合意
- 符合合同成立条件（第3条第1款）
- 未明确约定在将来一定期限内另行订立合同
- 或者一方已实施履行行为且对方接受

**判断逻辑**:
```
认购书/订购书等
    ↓
是否约定将来订立合同？
    ↓
  是 → 预约合同
    ↓
  否 → 已就主要内容达成合意 + 有履行行为？
    ↓
  是 → 本约合同
  否 → 预约合同
```

**示例**:
> 案情: 开发商与购房者签订认购书，约定房屋面积、单价、总价，但未约定在将来一定期限内另行订立合同，购房者已支付定金并接受房屋交付。

> 分析:
> - 根据第6条第3款，已就合同标的、数量、价款等主要内容达成合意
> - 且一方已实施履行行为并对方接受
> - → 认定本约合同成立
> - 不适用预约合同规则

#### 2.2 违反预约合同的认定（第7条）

**构成违反预约合同的情形**:

1. **明确拒绝订立本约合同**
2. **磋商时违背诚信原则导致未能订立本约合同**

**认定违背诚信原则的因素**:
- 磋商时提出的条件是否明显背离预约合同约定的内容
- 是否已尽合理努力进行协商
- 是否拒绝讨价还价
- 报价是否明显高出或低于市场价格

#### 2.3 违反预约合同的违约责任（第8条）

**不强制履行**: 不强制订立本约合同

**赔偿范围酌定因素（第8条第2款）**:
- 预约合同在内容上的完备程度
- 订立本约合同的条件的成就程度
- 交易的成熟度

**赔偿标准**:
- 交易成熟度较低 → 接近信赖利益
- 交易成熟度较高 → 接近履行利益
- **已经就本约合同的主要内容进行全面约定，且不存在阻碍本约合同订立的情形 → 可以参照本约合同的履行利益**

### 升级点3：批准生效合同的法律适用 ⭐⭐⭐⭐⭐

#### 3.1 报批义务的独立生效（第12条）

**合同依法成立后**:

负有报批义务的当事人不履行报批义务 → 对方可以请求：
- 继续履行报批义务
- 解除合同并请求赔偿违反报批义务的责任

#### 3.2 判决履行后仍不履行的后果

人民法院判决当事人一方履行报批义务后，其仍不履行：
- 对方可以主张解除合同
- 并请求参照违反合同的违约责任（赔偿全部履行利益）

#### 3.3 合同未获批准的后果

**情形1**: 报批义务人全面履行报批义务，但审批机关未批准
- 报批义务人不承担民事责任
- 允许当事人解除合同

**情形2**: 报批义务人履行报批义务不符合法律规定或合同约定
- 承担缔约过失责任（民法典第157条）

#### 3.4 程序规则

合同获得批准前，当事人一方起诉请求对方履行合同约定的主要义务：
- 经释明后拒绝变更诉讼请求 → 判决驳回诉讼请求
- 但不影响其另行提起诉讼

### 升级点4：无权处分的合同效力 ⭐⭐⭐⭐⭐

#### 4.1 无权处分不影响合同效力（第19条）

**以转让或者设定财产权利为目的订立的合同**:

当事人或真正权利人仅以让与人无所有权或处分权为由主张合同无效
- **人民法院不予支持**

#### 4.2 合同有效但不能履行的后果

因未取得真正权利人事后同意或让与人事后未取得处分权导致合同不能履行：
- 受让人可以主张：
  - 解除合同
  - 请求让与人承担违反合同的赔偿责任

#### 4.3 善意取得的衔接

合同有效且让与人已将财产交付或移转登记至受让人：
- 真正权利人请求：
  - 认定财产权利未发生变动
  - 或者请求返还财产
- 人民法院应予支持
- **但书**: 受让人依据民法典第311条善意取得财产权利的除外

#### 4.4 类推适用

无权处分所订买卖合同的规则，应类推适用到：
- 无权处分所订其他合同（抵押合同、质押合同等）
- 转让他人的其他财产权利
- 在他人的其他财产权利上设立担保物权

### 升级点5：越权代表与职务代理 ⭐⭐⭐⭐⭐

#### 5.1 越权代表（第20条）

**相对人合理审查义务**:

法律、行政法规规定合同所涉事项应当由法人、非法人组织的权力机构或决策机构决议：
- 法定代表人、负责人未取得授权
- 相对人未尽到合理审查义务 → 合同对法人、非法人组织不发生效力
- 法人、非法人组织有过错 → 参照民法典第157条承担相应的赔偿责任

**表见代表**:
- 相对人已尽到合理审查义务，构成表见代表
- 依据民法典第504条处理（合同有效）

**章程限制的对外效力**:
- 合同所涉事项未超越法律、行政法规规定的代表权限
- 但超越法人、非法人组织的章程或权力机构对代表权的限制
- 相对人主张合同有效 → 人民法院予以支持
- **但书**: 法人、非法人组织举证证明相对人知道或者应当知道该限制

**追偿权**: 法人、非法人组织承担民事责任后 → 向有过错的法定代表人、负责人追偿

**示例**:
> 案情: 公司法定代表人未经股东会决议对外提供担保，银行未审查决议。

> 分析:
> - 根据《民法典担保制度司法解释》第7条，相对人应审查决议
> - 银行未尽到合理审查义务
> - → 合同对公司不发生效力
> - 公司有过错 → 承担相应的赔偿责任（缔约过失责任，按份责任，不超过债务的三分之一）

#### 5.2 职务代理（第21条）

**超越职权范围的认定**:

合同所涉事项有下列情形之一的，认定工作人员超越职权范围：
1. 依法应当由权力机构或决策机构决议的事项
2. 依法应当由执行机构决定的事项
3. 依法应当由法定代表人、负责人实施的事项
4. 不属于通常情形下依其职权可以处理的事项

**法律后果**:
- 相对人主张合同有效 → 人民法院不予支持
- 法人、非法人组织有过错 → 承担相应的赔偿责任
- 构成表见代理 → 依据民法典第172条处理

**内部限制的对外效力**:
- 未超越依据前款确定的职权范围，但超越法人、非法人组织对工作人员职权范围的限制
- 相对人主张合同有效 → 人民法院予以支持
- **但书**: 法人、非法人组织举证证明相对人知道或者应当知道该限制

#### 5.3 印章与合同效力（第22条）

**核心规则**:
- 未超越权限 + 加盖非备案印章或伪造印章 → 合同有效
- 未加盖印章 + 有签名/按指印 + 相对人能证明未超越权限 → 合同有效
- 仅加盖印章 + 无人签名/按指印 + 相对人能证明在权限范围内订立 → 合同有效
- 虽超越权限，但构成表见代表或表见代理 → 合同有效

**例外**: 当事人约定以加盖印章作为合同成立条件

### 升级点6：格式条款的认定与提示义务 ⭐⭐⭐⭐

#### 6.1 格式条款的认定（第9条）

**核心规则**:
- 仅以合同系依据合同示范文本制作为由 → 不予支持
- 仅以双方明确约定合同条款不属于格式条款为由 → 不予支持
- 以未实际重复使用为由主张不是格式条款 → 不予支持
- **但书**: 有证据证明该条款不是为了重复使用而预先拟定的除外

#### 6.2 提示义务和说明义务（第10条）

**提示义务的履行**:
- 采用通常足以引起对方注意的明显标识：
  - 文字、符号、字体等
- 提示注意免除或者减轻其责任、排除或者限制对方权利等与对方有重大利害关系的异常条款

**说明义务的履行**:
- 按照对方要求，以书面或者口头形式作出通常能够理解的解释说明

**电子合同的特殊规则**:
- 通过互联网等信息网络订立的电子合同
- 仅以采取了设置勾选、弹窗等方式为由主张已履行提示义务或说明义务 → 不予支持
- 但其举证符合前两款规定的除外

**举证责任**: 提供格式条款的一方对已尽到提示义务或说明义务承担举证责任

### 升级点7：以物抵债协议 ⭐⭐⭐⭐

#### 7.1 债务履行期限届满后的以物抵债协议（第27条）

**效力**: 自当事人意思表示一致时生效

**履行**:
- 履行以物抵债协议 → 相应的原债务同时消灭
- 未按约定履行，经催告后在合理期限内仍不履行 → 债权人选择请求履行原债务或以物抵债协议

**物权变动**:
- 以物抵债协议经人民法院确认或制作成调解书
- 债权人主张财产权利自确认书、调解书生效时发生变动或具有对抗善意第三人效力 → **不予支持**

**无权处分**:
- 债务人或第三人以自己不享有所有权或处分权的财产权利订立以物抵债协议
- 依据第19条处理（无权处分规则）

#### 7.2 债务履行期限届满前的以物抵债协议（第28条）

**认定**: 在审理债权债务关系的基础上认定协议效力

**担保型以物抵债**:
- 约定债务人到期没有清偿债务，债权人可以对抵债财产拍卖、变卖、折价以实现债权 → **有效**

**流质契约禁止**:
- 约定债务人到期没有清偿债务，抵债财产归债权人所有 → **无效**
- 但不影响其他部分的效力
- 债权人请求对抵债财产拍卖、变卖、折价以实现债权 → 予以支持

**优先受偿**:
- 订立协议后，财产权利未转移至债权人 → 债权人主张优先受偿 → **不予支持**
- 财产权利已转移至债权人 → 依据《民法典担保制度司法解释》第68条处理

### 升级点8：代位权与撤销权 ⭐⭐⭐⭐

#### 8.1 代位权（第33-41条）

**"怠于行使"的认定（第33条）**:
- 债务人不履行其对债权人的到期债务
- 又不以诉讼或仲裁方式向相对人主张其享有的债权或与该债权有关的从权利
- 致使债权人的到期债权未能实现

**专属于债务人自身的权利（第34条）**:
- 抚养费、赡养费或扶养费请求权
- 人身损害赔偿请求权
- 劳动报酬请求权（超过债务人及其所扶养家属的生活必需费用的部分除外）
- 请求支付基本养老保险金、失业保险金、最低生活保障金等保障当事人基本生活的权利
- 其他专属于债务人自身的权利

**管辖（第35条）**: 被告住所地人民法院管辖（专属管辖除外）

**仲裁协议的效力（第36条）**:
- 债务人或相对人以仲裁协议为由提出异议 → 不予支持
- 但在首次开庭前申请仲裁的 → 可以中止代位权诉讼

**当事人（第37条）**:
- 应追加债务人为第三人
- 两个以上债权人提起代位权诉讼 → 可以合并审理
- 债权人对相对人的债权不足以清偿其对两个以上债权人负担的债务 → 按债权比例确定履行份额

#### 8.2 撤销权（第42-46条）

**"明显不合理"的认定（第42条）**:
- 转让价格未达到市场交易价或指导价的70% → "明显不合理的低价"
- 受让价格高于市场交易价或指导价的30% → "明显不合理的高价"
- 债务人与相对人存在亲属关系、关联关系 → 不受70%、30%的限制

**适用范围（第43条）**:
- 互易财产、以物抵债、出租或承租财产、知识产权许可使用等行为

**管辖（第44条）**: 债务人或相对人的住所地人民法院管辖（专属管辖除外）

**法律效果（第46条）**:
- 债权人可以请求债务人的相对人向债务人承担：
  - 返还财产
  - 折价补偿
  - 履行到期债务等法律后果

### 升级点9：合同解除 ⭐⭐⭐⭐

#### 9.1 协商解除（第52条）

**就解除合同协商一致但未对违约责任、结算和清理等问题作出处理**:
- 一方主张合同已经解除 → 予以支持
- 违约责任、结算和清理等问题 → 依据民法典第566条、第567条和有关违约责任的规定处理

**默示解除（第52条第2款）**:
- 一方主张行使解除权，经审理认为不符合解除权行使条件但是对方同意解除
- 双方当事人均不符合解除权行使条件但均主张解除合同

#### 9.2 解除权的行使（第53条）

**通知方式解除**:
- 以对方未在约定的异议期限或其他合理期限内提出异议为由主张合同已经解除
- 人民法院应对是否享有解除权进行审查
- 享有解除权 → 合同自通知到达对方时解除
- 不享有解除权 → 不发生合同解除的效力

#### 9.3 起诉解除（第54条）

**未通知对方，直接以提起诉讼的方式主张解除合同**:
- 撤诉后再次起诉主张解除合同，经审理支持
- → 合同自再次起诉的起诉状副本送达对方时解除
- **但书**: 撤诉后又通知对方解除合同且该通知已经到达对方的除外

---

### 升级点10：违约责任 ⭐⭐⭐⭐⭐

#### 10.1 可得利益的计算（第60条）

**扣除合理成本后计算**:
- 非违约方能够获得的生产利润、经营利润或转售利润等

**替代交易（第60条第2款）**:
- 实施了替代交易 → 按照替代交易价格与合同价格的差额确定
- 替代交易价格明显偏离市场价格 → 按照市场价格与合同价格的差额确定

**未实施替代交易（第60条第3款）**:
- 按照违约行为发生后合理期间内合同履行地的市场价格与合同价格的差额确定

**定期合同（第61条）**:
- 参考合同主体、交易类型、市场价格变化、剩余履行期限等因素确定非违约方寻找替代交易的合理期限
- 按照该期限对应的价款、租金等扣除非违约方应当支付的相应履约成本确定
- **不支持**: 按照合同解除后剩余履行期限相应的价款、租金等扣除履约成本确定（但剩余履行期限少于寻找替代交易的合理期限的除外）

**难以确定（第62条）**:
- 综合考虑违约方因违约获得的利益、违约方的过错程度、其他违约情节等因素
- 遵循公平原则和诚信原则确定

**示例**:
> 案情: 甲公司向乙公司购买设备用于生产，乙公司延迟交付导致甲公司停产。

> 分析:
> - 根据第60条第1款，计算可得利益：能够获得的生产利润
> - 扣除非违约方为订立、履行合同支出的费用等合理成本
> - 如果实施了替代交易 → 按照替代交易价格与合同价格的差额确定

#### 10.2 可预见性规则（第63条）

**认定"预见到或者应当预见到的损失"**:
- 根据当事人订立合同的目的
- 综合考虑合同主体、合同内容、交易类型、交易习惯、磋商过程等因素
- 按照与违约方处于相同或类似情况的民事主体在订立合同时预见到或应当预见到的损失予以确定

**额外费用**:
- 向第三人承担违约责任应当支出的额外费用等其他因违约所造成的损失
- 经审理认为系违约一方订立合同时预见到或应当预见到的 → 予以支持

#### 10.3 违约金调整（第64-66条）

**举证责任**:
- 违约方主张违约金过分高于违约造成的损失 → 承担举证责任
- 非违约方主张违约金合理 → 也应当提供相应的证据

**调整标准（第65条）**:
- 以民法典第584条规定的损失为基础
- 兼顾合同主体、交易类型、合同的履行情况、当事人的过错程度、履约背景等因素
- 遵循公平原则和诚信原则
- **约定的违约金超过造成损失的30% → 一般认定为过分高于造成的损失**
- **恶意违约**: 恶意违约的当事人一方请求减少违约金 → 一般不予支持

**二审释明（第66条）**:
- 一审认为抗辩成立且未予释明，二审认为应当判决支付违约金 → 可以直接释明
- 被告因客观原因在一审程序中未到庭参加诉讼，但在二审程序中到庭参加诉讼并请求减少违约金 → 可以依法判决适当减少违约金

**示例**:
> 案情: 合同约定违约金为合同总额的50%，违约方主张过分高于实际损失。

> 分析:
> - 根据第65条，约定的违约金超过造成损失的30% → 一般认定为过分高于造成的损失
> - 以民法典第584条规定的损失为基础
> - 兼顾合同主体、交易类型、合同的履行情况、当事人的过错程度、履约背景等因素
> - 如果是恶意违约 → 一般不予支持减少

#### 10.4 定金适用（第67-68条）

**定金性质的认定（第67条）**:
- 交付留置金、担保金、保证金、订约金、押金或订金等
- 没有约定定金性质 → 主张适用定金罚则 → 不予支持
- 约定了定金性质但未约定定金类型或约定不明 → 主张为违约定金 → 予以支持

**立约定金**:
- 约定以交付定金作为订立合同的担保
- 一方拒绝订立合同或在磋商订立合同时违背诚信原则导致未能订立合同 → 可以适用定金罚则

**成约定金（第67条第3款）**:
- 约定以交付定金作为合同成立或生效条件
- 应交付定金的一方未交付定金，但合同主要义务已经履行完毕并为对方所接受
- → 认定合同在对方接受履行时已经成立或生效

**解约定金（第67条第4款）**:
- 约定定金性质为解约定金
- 交付定金的一方主张以丧失定金为代价解除合同，或收受定金的一方主张以双倍返还定金为代价解除合同
- → 予以支持

**双方违约（第68条第1款）**:
- 双方当事人均具有致使不能实现合同目的的违约行为
- 一方请求适用定金罚则 → 不予支持

**轻微违约（第68条第2款）**:
- 一方仅有轻微违约，对方具有致使不能实现合同目的的违约行为
- 轻微违约方主张适用定金罚则 → 对方以轻微违约方也构成违约为由抗辩 → 对该抗辩不予支持

**部分履行（第68条第3款）**:
- 一方已经部分履行合同，对方接受
- 主张按照未履行部分所占比例适用定金罚则 → 予以支持
- **但书**: 部分未履行致使不能实现合同目的的除外

**不可抗力（第68条第4款）**:
- 因不可抗力致使合同不能履行 → 非违约方主张适用定金罚则 → 不予支持

## 三、典型案例与应用场景

### 案例1：预约合同与本约合同的区分

**案情**: 开发商与购房者签订认购书，约定房屋面积、单价、总价，但未约定在将来一定期限内另行订立合同，购房者已支付定金并接受房屋交付。

**分析要点**:
- 根据第6条第3款，已就合同标的、数量、价款等主要内容达成合意，且一方已实施履行行为并对方接受
- → 认定本约合同成立
- 不适用预约合同规则

---

### 案例2：越权代表的认定

**案情**: 公司法定代表人未经股东会决议对外提供担保，银行未审查决议。

**分析要点**:
- 根据《民法典担保制度司法解释》第7条，相对人应审查决议
- 银行未尽到合理审查义务
- → 合同对公司不发生效力
- 公司有过错 → 承担相应的赔偿责任（缔约过失责任，按份责任，不超过债务的三分之一）

---

### 案例3：违反强制性规定但不导致合同无效

**案情**: 开发商未取得预售许可证明即签订商品房买卖合同，房价上涨后开发商以合同无效为由拒绝履行。

**分析要点**:
- 根据《商品房买卖合同司法解释》第2条，起诉前未取得预售许可证明 → 合同无效
- 但如果起诉前已取得预售许可证明 → 合同有效
- 如果开发商已具备申请条件但违背诚信原则不申请 → 违背诚信原则，不应支持

---

### 案例4：可得利益损失的计算

**案情**: 甲公司向乙公司购买设备用于生产，乙公司延迟交付导致甲公司停产。

**分析要点**:
- 根据第60条第1款，计算可得利益：能够获得的生产利润
- 扣除非违约方为订立、履行合同支出的费用等合理成本
- 如果实施了替代交易 → 按照替代交易价格与合同价格的差额确定

---

### 案例5：违约金调整

**案情**: 合同约定违约金为合同总额的50%，违约方主张过分高于实际损失。

**分析要点**:
- 根据第65条，约定的违约金超过造成损失的30% → 一般认定为过分高于造成的损失
- 以民法典第584条规定的损失为基础
- 兼顾合同主体、交易类型、合同的履行情况、当事人的过错程度、履约背景等因素
- 如果是恶意违约 → 一般不予支持减少
