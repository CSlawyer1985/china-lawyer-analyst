## 使用说明

### 触发机制
每个领域模块都有特定的触发关键词,当用户提问包含这些关键词时,自动加载相应的领域模块。

### 模块化设计
- 每个模块都是独立的、完整的
- 可以单独使用,也可以组合使用
- 每个模块都依赖核心模块(core/philosophy.md, core/foundations-universal.md, core/frameworks-core.md)

### 引用说明
如果某个概念在其他模块有详细说明,会添加"详见 core/xxx.md"或"详见 domains/xxx.md"的引用。
