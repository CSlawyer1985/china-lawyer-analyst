## 项目简介

`china-lawyer-analyst` 是一个适用于中国大陆法律体系的专业法律分析 Claude Skill。该 skill 将美国普通法系的 `lawyer-analyst` 本土化改造，基于大陆法系（成文法）体系，运用中国法律方法论进行法律分析。

### v2.0 核心升级

**MOE 混合专家架构**：
- ✅ **智能路由**：100% 路由准确率，自动识别法律领域
- ✅ **按需加载**：根据问题类型动态加载相关模块
- ✅ **Token 优化**：平均节省 37% tokens（单领域节省 48%）
- ✅ **透明化反馈**：清晰告知加载了哪些模块

### 核心特点

- ✅ **成文法权威**：以宪法、法律、行政法规、司法解释为主要法律渊源
- ✅ **指导案例参照**：参照最高人民法院指导性案例
- ✅ **10步法流程**：系统化的中国法律分析流程
- ✅ **IRAC方法本土化**：结合三段论、请求权基础分析法
- ✅ **权威数据库**：威科先行、北大法宝、法信、裁判文书网
- ✅ **8大领域覆盖**：合同法、侵权法、建设工程、公司法、投融资、劳动法、知识产权、诉讼仲裁

### 应用场景

1. **律师实务工作**：合同审查、纠纷解决、风险评估、诉讼策略
2. **律所培训/知识管理**：标准化分析方法、案例库、知识沉淀
3. **学术研究/写作辅助**：法学论文、案例分析、法律注释
4. **Legal Tech产品开发**：法律AI、智能问答、文书生成

## 10步法中国法律分析流程

当使用 `china-lawyer-analyst` skill 时，系统将遵循以下10步流程：

### Step 1: 收集事实
- 识别关键事实
- 整理时间线
- 识别当事人

### Step 2: 识别法律问题
- 将事实转化为法律问题
- 识别争议焦点

### Step 3: 研究适用法律
- 查找成文法（法律、行政法规）
- 查找司法解释
- 参照指导案例

### Step 4: IRAC分析
- **Issue**：问题识别
- **Rule**：规则识别
- **Application**：适用分析
- **Conclusion**：结论

### Step 5: 识别诉讼请求和抗辩
- 确定我方诉讼请求（原告）或抗辩理由（被告）
- 评估证据支持

### Step 6: 评估程序阶段和选项
- 确定当前程序阶段（起诉前、一审、二审、再审、执行）
- 评估程序选项（财产保全、证据保全）

### Step 7: 评估风险和可能结果
- 胜诉概率评估
- 可能结果分析（最佳、最差、最可能）
- 成本收益分析

### Step 8: 识别和评估救济措施
- 继续履行、赔偿损失、违约金、定金
- 量化损害赔偿

### Step 9: 考虑和解和替代争议解决
- 评估 BATNA（最佳替代方案）
- 考虑人民调解、司法调解、商事仲裁

### Step 10: 提供法律建议和行动计划
- 综合分析，提供明确法律意见
- 提出策略建议
- 提出行动计划

## 常见误区与避免方法

### 误区 1: 忽视司法解释的效力

**错误做法**：只引用法律条文，忽视司法解释

**正确做法**：司法解释具有法律效力（经全国人大常委会备案），引用法律时应当同时查找相关司法解释

### 误区 2: 混淆指导案例与判例拘束力

**错误做法**：认为指导案例具有拘束力（如美国判例法）

**正确做法**：中国是成文法国家，指导案例具有"参照"作用，非"拘束"力

### 误区 3: 举证责任分配错误

**错误做法**：认为"谁主张谁举证"适用于所有情形

**正确做法**：一般原则是"谁主张谁举证"，但存在举证责任倒置的法定情形（环境污染、饲养动物、医疗损害等）

### 误区 4: 超过诉讼时效、除斥期间

**错误做法**：忽视诉讼时效、除斥期间

**正确做法**：
- 诉讼时效：一般3年（民法典第188条）
- 除斥期间：不变期间，不适用中止、中断、延长

更多常见误区详见 `shared/verification/pitfalls.md`

## 引用格式标准

### 法律条文引用

```
【法律类型】《法律名称》第X条第X款

示例：
- 【法律】《中华人民共和国民法典》第五百七十七条
- 【行政法规】《建设工程质量管理条例》第三条
- 【司法解释】《最高人民法院关于适用〈中华人民共和国民事诉讼法〉的解释》第九十条
```

### 案例引用

```
【法院全称】（XXXX）XX法XX终字XX号【民事判决书/裁定书】

示例：
- 最高人民法院（2019）最高法民终245号民事判决书
- 浙江省高级人民法院（2020）浙民终123号民事判决书
```
