## 数据库使用指南

### 1. 国家法律法规数据库

**适用场景**：查询法律、行政法规、司法解释的权威文本

**使用方法**：
1. 访问 https://flk.npc.gov.cn
2. 在搜索框输入司法解释名称或文号
3. 筛选"司法解释"类型
4. 查看全文、效力状态、历次修改

**优势**：
- ✅ 官方权威，文本准确
- ✅ 实时更新，时效性强
- ✅ 免费，无需注册

### 2. 中国裁判文书网

**适用场景**：查找司法解释在具体案件中的适用情况

**使用方法**：
1. 访问 https://wenshu.court.gov.cn
2. 高级检索 → 案由/法律依据
3. 输入司法解释条文编号或关键词
4. 查看相关裁判文书

**优势**：
- ✅ 官方平台，案例全面
- ✅ 可以看到条文在实践中的具体适用
- ✅ 了解各地法院的裁判观点

**注意事项**：
- ⚠️ 需要注册登录
- ⚠️ 文书量巨大，需要精准检索
- ⚠️ 部分敏感案件不公开

### 3. 威科先行

**适用场景**：深度学习司法解释，获取专业解读和实务指南

**使用方法**：
1. 访问 https://law.wkinfo.com.cn
2. 搜索司法解释名称或条文
3. 查看"实务指南""相关案例""学理观点"

**优势**：
- ✅ 条文解读专业深入
- ✅ 实务指南针对性强
- ✅ 案例关联准确

**注意事项**：
- ⚠️ 需要付费订阅
- ⚠️ 部分内容仅限机构用户

### 4. 北大法宝

**适用场景**：学术研究，获取法学理论和学术观点

**使用方法**：
1. 访问 https://www.pkulaw.com
2. 搜索司法解释条文
3. 查看"法学文献""期刊论文""学位论文"

**优势**：
- ✅ 学术资源丰富
- ✅ 法条联想智能
- ✅ 引文分析功能

**注意事项**：
- ⚠️ 需要付费订阅
- ⚠️ 部分功能仅限校园网

## 检索技巧

### 组合检索

**场景**：需要查找特定条文在特定地区的适用情况

**检索式**：
```
"第X条" + "保证方式约定不明" + "北京"
"第37条" + "抵押财产转让" + "上海市高级人民法院"
```

### 时间筛选

**场景**：需要查找最新案例

**检索式**：
```
"第25条" + "保证方式" + 裁判日期：2023-2024
```

### 案例类型筛选

**场景**：需要查找最高人民法院的案例

**检索式**：
```
"第7条" + "相对人善意" + 审级：最高人民法院
```

## 数据库选择决策树

```
需要查找司法解释内容
    ↓
仅需要条文全文？
    ├─ 是 → 国家法律法规数据库（免费、权威）
    └─ 否 → 需要解读或案例？
        ├─ 是 → 威科先行或北大法宝（专业、深入）
        └─ 否 → 需要案例？
            ├─ 是 → 中国裁判文书网（官方、全面）
            │        或人民法院案例库（精选、权威）
            └─ 否 → 需要学术观点？
                └─ 是 → 北大法宝（学术文献）
```
