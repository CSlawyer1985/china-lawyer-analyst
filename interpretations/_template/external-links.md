## 数据库使用指南

### 1. 国家法律法规数据库（⭐首选）

**适用场景**：查询法律、行政法规、司法解释的权威文本

**使用方法**：
1. 访问 https://flk.npc.gov.cn
2. 在搜索框输入司法解释名称或文号
3. 筛选"司法解释"类型
4. 查看全文、效力状态、历次修改

**优势**：
- ✅ 官方权威，文本准确
- ✅ 实时更新，时效性强
- ✅ 免费，无需注册

---

### 2. 人民法院案例库（⭐推荐）

**适用场景**：查找指导案例、典型案例、参考案例

**使用方法**：
1. 访问 https://anli.court.gov.cn
2. 在搜索框输入案件类型、法条或关键词
3. 筛选案例类型（指导案例、典型案例、参考案例）
4. 查看裁判要点、裁判理由

**优势**：
- ✅ 最高法院官方案例库
- ✅ 案例权威性强
- ✅ 裁判要点清晰
- ✅ 免费，无需注册

---

### 3. 中国裁判文书网（⭐推荐）

**适用场景**：查找司法解释在具体案件中的适用情况

**使用方法**：
1. 访问 https://wenshu.court.gov.cn
2. 高级检索 → 案由/法律依据
3. 输入司法解释条文编号或关键词
4. 查看相关裁判文书

**优势**：
- ✅ 官方平台，案例全面
- ✅ 可以看到条文在实践中的具体适用
- ✅ 了解各地法院的裁判观点
- ✅ 免费

**注意事项**：
- ⚠️ 需要注册登录
- ⚠️ 文书量巨大，需要精准检索
- ⚠️ 部分敏感案件不公开

---

### 4. 最高人民法院官网（⭐权威）

**适用场景**：查询最新司法解释、司法政策、指导案例

**使用方法**：
1. 访问 https://www.court.gov.cn
2. 进入"司法解释"或"指导案例"栏目
3. 按发布时间或分类浏览
4. 查看司法解释全文、理解与适用

**优势**：
- ✅ 最高法院官方发布
- ✅ 最权威、最及时
- ✅ 包含司法解释理解与适用
- ✅ 免费

## 检索技巧

### 组合检索

**场景**：需要查找特定条文在特定地区的适用情况

**检索式**：
```
"第X条" + "保证方式约定不明" + "北京"
"第37条" + "抵押财产转让" + "上海市高级人民法院"
```

### 时间筛选

**场景**：需要查找最新案例

**检索式**：
```
"第25条" + "保证方式" + 裁判日期：2023-2024
```

### 案例类型筛选

**场景**：需要查找最高人民法院的案例

**检索式**：
```
"第7条" + "相对人善意" + 审级：最高人民法院
```

## 数据库选择决策树（优先官方权威数据库）

```
需要查找司法解释内容
    ↓
仅需要条文全文？
    ├─ 是 → 国家法律法规数据库（免费、权威）⭐
    └─ 否 → 需要案例或解读？
        ├─ 需要权威案例 → 人民法院案例库（官方、精选）⭐

        └─ 需要学术观点 → 北大法宝（付费，学术研究）⚠️
```

**重要提示**：
- ⭐ 官方权威数据库：免费、权威、推荐优先使用
- ⚠️ 付费专业数据库：仅供深度学术研究时参考
