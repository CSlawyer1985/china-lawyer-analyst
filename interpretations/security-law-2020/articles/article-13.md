# 第13条：共同担保人之间的追偿与分担

---

## 条文内容

> 同一债务有两个以上第三人提供担保，担保人之间约定相互追偿及分担份额，承担了担保责任的担保人请求其他担保人按照约定分担份额的，人民法院应予支持；担保人之间约定承担连带共同担保，或者约定相互追偿但是未约定分担份额的，各担保人按照比例分担向债务人不能追偿的部分。
>
> 同一债务有两个以上第三人提供担保，担保人之间未对相互追偿作出约定且未约定承担连带共同担保，但是各担保人在同一份合同书上签字、盖章或者按指印，承担了担保责任的担保人请求其他担保人按照比例分担向债务人不能追偿部分的，人民法院应予支持。
>
> 除前两款规定的情形外，承担了担保责任的担保人请求其他担保人分担向债务人不能追偿部分的，人民法院不予支持。

---

## 核心要点

### 1. 共同担保的基本类型

**共同担保**：两个以上第三人为同一债务提供担保

**三种类型**：
1. **共同保证**：多个保证人提供保证
2. **共同物保**：多个第三人提供物的担保
3. **混合共同担保**：既有保证又有第三人提供的物的担保

### 2. 基本原则：担保人之间原则上不能相互追偿

**立法背景**：
- 《民法典》第392条、第700条删除了担保人之间相互追偿的规定
- 立法机关明确：担保人之间原则上不能相互追偿

**理由**：
1. **理论上讲不通**：各担保人提供的担保具有独立性，担保人之间相互没有意思联络
2. **程序上不经济**：向其他担保人追偿后，仍需向债务人追偿
3. **体现公平原则**：担保人在设定担保时，对于其面临的风险是明知的
4. **可操作性差**：份额确定和计算难以准确界定

### 3. 三种例外情形：可以相互追偿

#### 例外情形1：约定相互追偿及分担份额
- 担保人之间明确约定可以相互追偿
- 约定了分担份额
- 承担了担保责任的担保人可以请求其他担保人按照约定分担份额

#### 例外情形2：约定连带共同担保或约定相互追偿但未约定分担份额
- 约定承担连带共同担保
- 或者约定相互追偿但是未约定分担份额
- 各担保人按照比例分担向债务人不能追偿的部分

#### 例外情形3：在同一份合同书上签字、盖章或按指印
- 担保人之间未对相互追偿作出约定且未约定承担连带共同担保
- 但各担保人在同一份合同书上签字、盖章或者按指印
- 推定存在连带共同担保的意思联络
- 承担了担保责任的担保人可以请求其他担保人按照比例分担向债务人不能追偿的部分

## 深度实务要点

### 一、共同担保的类型区分

#### （一）按份共同担保

**特征**：
- 担保人在担保合同中明确约定各自应当承担的担保份额
- 各个担保人之间原则上没有牵连关系
- 其所承担的担保责任范围依据自己的承诺而产生

**法律后果**：
- 担保人履行担保责任后只能向债务人追偿
- 担保人之间不存在相互追偿的问题

**示例**：
```
债权人甲对债务人乙享有300万元债权
保证人A、B、C与债权人甲签订保证合同
约定各自承担100万元的保证责任
→ 按份共同保证
→ 保证人A承担100万元后，只能向债务人乙追偿100万元
→ 不能要求保证人B、C分担
```

#### （二）连带共同担保

**特征**：
- 多个担保人在担保合同中没有约定担保份额
- 各担保人均对债权人承担连带责任
- 债权人可以要求任何一个担保人承担全部担保责任

**分类**：
1. **真正的连带共同担保**：担保人之间存在相互追偿权
2. **不真正的连带共同担保**：担保人之间不存在相互追偿权

**法律后果**：
- 对外关系：各担保人均对债权人承担连带责任
- 对内关系：取决于是否存在例外情形

---

### 二、三种例外情形的认定标准

#### 例外情形1：约定相互追偿及分担份额

**认定标准**：
- 担保人之间明确约定可以相互追偿
- 约定了分担份额（可以是具体金额，也可以是比例）

**典型表述**：
- "担保人之间可以相互追偿"
- "各担保人按照XX比例分担"
- "承担担保责任的担保人有权要求其他担保人分担其应承担的份额"

**法律后果**：
- 承担了担保责任的担保人可以请求其他担保人按照约定分担份额
- 不需要先向债务人追偿
- 可以直接要求其他担保人分担

#### 例外情形2：约定连带共同担保或约定相互追偿但未约定分担份额

**认定标准**：
- 约定承担连带共同担保
- 或者约定相互追偿但是未约定分担份额

**典型表述**：
- "各担保人承担连带共同担保责任"
- "担保人之间可以相互追偿"（但未约定分担比例）

**法律后果**：
- 应当先向主债务人追偿
- 对于主债务人不能追偿的部分，再向其他担保人追偿
- 避免循环追偿问题

**示例**：
```
保证人A、B、C约定承担连带共同担保责任
债务人乙无力清偿债务，甲债权人要求保证人A承担300万元
保证人A清偿300万元后
→ 第一步：向债务人乙追偿300万元
→ 第二步：债务人乙不能清偿的部分（如200万元），由保证人A、B、C按照比例分担
→ 如果债务人乙完全无力清偿，保证人A可以要求B、C各分担100万元
```

#### 例外情形3：在同一份合同书上签字、盖章或按指印

**认定标准**：
- 担保人之间未对相互追偿作出约定且未约定承担连带共同担保
- 但各担保人在同一份合同书上签字、盖章或者按指印

**推定理由**：
- 可以理解为担保人之间存在连带共同担保的意思联络

**法律后果**：
- 推定存在连带共同担保
- 承担了担保责任的担保人可以请求其他担保人按照比例分担向债务人不能追偿的部分

**示例**：
```
保证人A、B、C在同一份保证合同上签字
合同中未约定担保人之间可以相互追偿
也未约定承担连带共同担保
→ 推定存在连带共同担保的意思联络
→ 保证人A承担300万元后，就向债务人不能追偿的部分，可以要求B、C分担
```

### 三、担保人分担份额比例的确定

#### （一）均为保证人的情形

**情形1：保证人均对主债权及其利息、违约金、损害赔偿金和实现债权的费用提供担保**
- 分担比例：**平均分担**
- 承担了保证责任的保证人，就向债务人不能追偿的部分，可以要求其他保证人均分

**示例**：
```
主债权：300万元及利息、违约金等
保证人A、B、C均提供担保
保证人A承担了全部责任
→ 就向债务人不能追偿的部分，可以要求B、C各分担1/3
```

**情形2：部分保证人对其承担的保证责任数额进行限制**
- 分担比例：按照保证人承担的保证债务数额为基础依比例计算

**示例**：
```
主债权：300万元
保证人A担保100万元
保证人B担保150万元
保证人C担保300万元
→ 分担比例：A:B:C = 1:1.5:3
→ 如果保证人C承担300万元后，就向债务人不能追偿的部分
→ 可以要求A分担1/6.5，B分担1.5/6.5
```

#### （二）混合担保的情形（保证+物上保证人）

**基本原则**：以各担保人承担的具体担保责任为基础比例计算

**情形1：担保财产的价值 < 担保债权额**
- 分担比例：按照保证债权额和担保财产价值的比例确定各自的份额

**示例**：
```
主债权：300万元
保证人A担保300万元
抵押人B提供价值150万元的抵押物
→ 分担比例：A:B = 300:150 = 2:1
→ 如果保证人A承担300万元后，就向债务人不能追偿的部分
→ 可以要求B分担1/3
```

**情形2：担保财产的价值 ≥ 担保债权额**
- 分担比例：以保证债权额和担保债权额的比例确定各自的份额

**示例**：
```
主债权：300万元
保证人A担保300万元
抵押人B提供价值500万元的抵押物（担保债权额300万元）
→ 分担比例：A:B = 300:300 = 1:1
→ 如果保证人A承担300万元后，就向债务人不能追偿的部分
→ 可以要求B分担1/2
```

#### （三）担保人均为物上保证人的情形

**情形1：担保财产价值均 < 担保债权额**
- 分担比例：根据各自担保财产的价值比例确定各自的份额

**示例**：
```
主债权：300万元
抵押人A提供价值150万元的抵押物
抵押人B提供价值100万元的抵押物
→ 分担比例：A:B = 150:100 = 3:2
```

**情形2：担保财产价值均 ≥ 担保债权额**
- 分担比例：根据各自担保债权额的比例确定分担的份额

**示例**：
```
主债权：300万元
抵押人A提供价值500万元的抵押物（担保债权额300万元）
抵押人B提供价值400万元的抵押物（担保债权额300万元）
→ 分担比例：A:B = 300:300 = 1:1
```

**情形3：部分担保财产价值 < 担保债权额**
- 分担比例：根据该担保人的担保财产价值和其他担保人担保债权额之间的比例确定各自分担的份额

**示例**：
```
主债权：300万元
抵押人A提供价值150万元的抵押物（担保债权额300万元）
抵押人B提供价值500万元的抵押物（担保债权额300万元）
→ 分担比例：A:B = 150:300 = 1:2
```

### 四、担保人追偿权的性质及范围

#### （一）追偿权的性质

**担保人承担担保责任后**：
- 原债权债务关系消灭
- 有权向债务人追偿
- **但**：担保人享有的追偿权和代位求偿权并不相同
- 担保人并不能代替原债权人享有原债权债务中原债权人的权利
- 只能在承担担保责任的范围内向债务人追偿

#### （二）追偿权的范围

**先向债务人追偿**：
- 承担了担保责任的担保人，应当先向债务人追偿
- 对于向债务人不能追偿的部分，才能向其他担保人追偿

**代位权**：
- 同一债权人有债务人自己提供的物的担保
- 承担了担保责任的担保人在其承担责任的范围内
- 主张行使债权人享有的担保物权的，人民法院应予支持

---

### 五、实务争议与处理

#### 争议焦点1：担保人能否通过受让债权的方式规避追偿限制？

**典型情形**：
- 担保人通过与债权人签订债权转让合同的方式
- 在受让债权后以债权人的身份向债务人以及其他担保人追偿
- 该行为试图规避《民法典》有关担保人之间追偿原则的规定

**处理方式**：
- 根据《民法典》第557条第1款第5项的规定，债权债务同归一人时债权债务终止
- 担保人受让债权的行为仅发生担保责任消灭或者部分消灭的效果
- 其行为本质上属于向债权人承担担保责任
- 担保人无权以债权人的身份请求其他担保人向其承担担保责任

**法律依据**：《民法典担保制度解释》第14条

#### 争议焦点2：按份共同担保中，担保人受让债权后能否要求其他担保人承担责任？

**处理方式**：
- 担保人的担保责任在其受让债权的范围内消灭
- 其他担保人对于债权人的责任不受影响
- 担保人无权基于债权转让合同要求其他担保人承担保证责任

**示例**：
```
债权人甲对债务人乙享有300万元债权
保证人A、B、C与债权人甲签订保证合同
约定各自承担100万元的保证责任
保证人A受让100万元债权
→ 保证人A的担保责任消灭
→ 保证人B和C仍然需要向债权人甲承担约定的保证责任
```

#### 争议焦点3：连带共同担保中，担保人受让债权后能否要求其他担保人承担责任？

**处理方式**：
- 根据担保人之间是否存在相互追偿权，区分处理

**情形1：真正的连带共同担保（担保人之间可以相互追偿）**
- 担保人的担保责任和受让债权因混同而消灭
- 对于超出其应承担的担保责任的部分
- 有权向其他担保人追偿，并享有债权人的权利
- 可以按照《民法典担保制度解释》第13条的规定
- 就向债务人不能追偿的部分，要求其他担保人分担

**情形2：不真正的连带共同担保（担保人之间不能相互追偿）**
- 担保责任和受让债权因混同而消灭
- 所担保的主债务也随之消灭
- 其他保证人无须承担担保责任

## 与旧法的区别

| 项目 | 旧法（担保法解释） | 新法（民法典+本条） |
|------|-------------------|-------------------|
| 担保人之间追偿权 | 原则上可以相互追偿 | 原则上不能相互追偿 |
| 例外情形 | 未规定 | 明确3种例外情形 |
| 分担份额确定 | 未明确 | 明确按比例确定 |
| 先追偿顺序 | 未明确 | 明确先向债务人追偿 |

## 实务应用

### 案例1：约定相互追偿及分担份额

**案情**：
甲公司对乙公司享有300万元债权。丙公司、丁公司、戊公司与甲公司签订保证合同，约定为乙公司的债务提供连带责任保证，同时约定："各保证人之间可以相互追偿，按照1:1:1的比例分担。"后乙公司无力清偿债务，甲公司要求丙公司承担300万元保证责任。丙公司清偿300万元后，向乙公司追偿，乙公司仅偿还100万元，剩余200万元无力偿还。丙公司起诉要求丁公司、戊公司各分担66.67万元。

**分析**：
1. 丙公司、丁公司、戊公司约定相互追偿及分担份额 ✅
2. 约定按照1:1:1的比例分担 ✅
3. 丙公司承担了300万元保证责任 ✅
4. 根据《民法典担保制度解释》第13条第1款：丙公司可以请求丁公司、戊公司按照约定分担份额 ✅

**结论**：丙公司可以请求丁公司、戊公司各分担66.67万元。

---

### 案例2：约定连带共同担保但未约定分担份额

**案情**：
甲公司对乙公司享有300万元债权。丙公司、丁公司、戊公司与甲公司签订保证合同，约定为乙公司的债务提供连带责任保证，同时约定："各保证人之间承担连带共同担保责任。"后乙公司无力清偿债务，甲公司要求丙公司承担300万元保证责任。丙公司清偿300万元后，向乙公司追偿，乙公司仅偿还100万元，剩余200万元无力偿还。丙公司起诉要求丁公司、戊公司分担。

**分析**：
1. 丙公司、丁公司、戊公司约定承担连带共同担保 ✅
2. 未约定分担份额 ✅
3. 丙公司承担了300万元保证责任 ✅
4. 丙公司先向乙公司追偿，乙公司仅偿还100万元 ✅
5. 向债务人不能追偿的部分：200万元 ✅
6. 根据《民法典担保制度解释》第13条第1款：各担保人按照比例分担向债务人不能追偿的部分 ✅
7. 分担比例：1:1:1 ✅

**结论**：丙公司可以请求丁公司、戊公司各分担66.67万元。

---

### 案例3：在同一份合同书上签字

**案情**：
甲公司对乙公司享有300万元债权。丙公司、丁公司、戊公司在同一份保证合同上签字，为乙公司的债务提供连带责任保证。合同中未约定保证人之间可以相互追偿，也未约定承担连带共同担保。后乙公司无力清偿债务，甲公司要求丙公司承担300万元保证责任。丙公司清偿300万元后，向乙公司追偿，乙公司仅偿还100万元，剩余200万元无力偿还。丙公司起诉要求丁公司、戊公司分担。

**分析**：
1. 丙公司、丁公司、戊公司在同一份合同书上签字 ✅
2. 未约定相互追偿 ✅
3. 未约定承担连带共同担保 ✅
4. 根据《民法典担保制度解释》第13条第2款：推定存在连带共同担保的意思联络 ✅
5. 丙公司就向债务人不能追偿的部分（200万元），可以要求丁公司、戊公司分担 ✅
6. 分担比例：1:1:1 ✅

**结论**：丙公司可以请求丁公司、戊公司各分担66.67万元。

---

### 案例4：混合担保的分担比例

**案情**：
甲公司对乙公司享有300万元债权。丙公司与甲公司签订保证合同，为乙公司的债务提供300万元保证责任。丁公司以其价值150万元的房产为乙公司的债务提供抵押担保，并办理抵押登记。合同中约定："担保人之间承担连带共同担保责任。"后乙公司无力清偿债务，甲公司要求丙公司承担300万元保证责任。丙公司清偿300万元后，向乙公司追偿，乙公司仅偿还100万元，剩余200万元无力偿还。丙公司起诉要求丁公司分担。

**分析**：
1. 丙公司、丁公司约定承担连带共同担保 ✅
2. 担保财产的价值（150万元）< 担保债权额（300万元）✅
3. 丙公司就向债务人不能追偿的部分（200万元），可以要求丁公司分担 ✅
4. 分担比例：保证债权额（300万元）:担保财产价值（150万元）= 2:1 ✅

**结论**：丙公司可以请求丁公司分担66.67万元（200万元 × 1/3）。

## 争议焦点

**焦点1：担保人之间能否通过受让债权的方式规避追偿限制？**

**观点**：
- 担保人受让债权的行为本质上属于承担担保责任
- 担保人无权以债权人的身份请求其他担保人向其承担担保责任
- 应当根据担保人之间是否存在相互追偿权，按照《民法典担保制度解释》第13条的规定处理

**焦点2：按份共同担保中，担保人受让债权后能否要求其他担保人承担责任？**

**观点**：
- 担保人的担保责任在其受让债权的范围内消灭
- 其他担保人对于债权人的责任不受影响
- 担保人无权基于债权转让合同要求其他担保人承担保证责任

**焦点3：连带共同担保中，担保人受让债权后能否要求其他担保人承担责任？**

**观点**：
- 根据担保人之间是否存在相互追偿权，区分处理
- 真正的连带共同担保：可以就向债务人不能追偿的部分，要求其他担保人分担
- 不真正的连带共同担保：其他保证人无须承担担保责任

## 法律后果

### 担保人之间可以相互追偿

**适用情形**：
1. 约定相互追偿及分担份额
2. 约定连带共同担保或约定相互追偿但未约定分担份额
3. 在同一份合同书上签字、盖章或按指印

**法律后果**：
- 承担了担保责任的担保人可以请求其他担保人分担
- 应当先向债务人追偿
- 对于向债务人不能追偿的部分，再向其他担保人追偿

### 担保人之间不能相互追偿

**适用情形**：
- 除上述三种例外情形外的其他情形

**法律后果**：
- 承担了担保责任的担保人只能向债务人追偿
- 无权请求其他担保人分担

## 相关条文

**《民法典》第392条**：
被担保的债权既有物的担保又有人的担保的，债务人不履行到期债务或者发生当事人约定的实现担保物权的情形，债权人应当按照约定实现债权；没有约定或者约定不明确，债务人自己提供物的担保的，债权人应当先就该物的担保实现债权；第三人提供物的担保的，债权人可以就物的担保实现债权，也可以请求保证人承担保证责任。提供担保的第三人承担担保责任后，有权向债务人追偿。

**《民法典》第519条**：
连带债务人之间的份额难以确定的，视为份额相同。实际承担债务超过自己份额的连带债务人，有权就超出部分在其他连带债务人未履行的份额范围内向其追偿，并相应地享有债权人的权利，但是不得损害债权人的利益。

**《民法典》第557条第1款第5项**：
债权债务同归一人时债权债务终止。

**《民法典》第699条**：
同一债务有两个以上保证人的，保证人应当按照保证合同约定的保证份额，承担保证责任；没有约定保证份额的，债权人可以请求任何一个保证人在其保证范围内承担保证责任。

**《民法典》第700条**：
保证人承担保证责任后，除当事人另有约定外，有权在其承担保证责任的范围内向债务人追偿，享有债权人对债务人的权利，但是不得损害债权人的利益。

**《民法典担保制度解释》第13条**：
同一债务有两个以上第三人提供担保，担保人之间约定相互追偿及分担份额，承担了担保责任的担保人请求其他担保人按照约定分担份额的，人民法院应予支持；担保人之间约定承担连带共同担保，或者约定相互追偿但是未约定分担份额的，各担保人按照比例分担向债务人不能追偿的部分。

同一债务有两个以上第三人提供担保，担保人之间未对相互追偿作出约定且未约定承担连带共同担保，但是各担保人在同一份合同书上签字、盖章或者按指印，承担了担保责任的担保人请求其他担保人按照比例分担向债务人不能追偿部分的，人民法院应予支持。

除前两款规定的情形外，承担了担保责任的担保人请求其他担保人分担向债务人不能追偿部分的，人民法院不予支持。

**《民法典担保制度解释》第14条**：
同一债务有两个以上第三人提供担保，担保人受让债权的，人民法院应当认定该行为系承担担保责任。受让债权的担保人作为债权人请求其他担保人承担担保责任的，人民法院不予支持；该担保人请求其他担保人分担相应份额的，依照本解释第十三条的规定处理。

## 实务建议

**1. 债权人（银行、金融机构等）**
- 审查担保合同中担保人之间的约定
- 如希望担保人之间可以相互追偿，应当在合同中明确约定
- 约定连带共同担保或约定相互追偿及分担份额
- 要求各担保人在同一份合同书上签字、盖章或者按指印

**2. 担保人**
- 明确担保人之间是否可以相互追偿
- 如希望担保人之间可以相互追偿，应当在合同中明确约定
- 约定分担份额以避免争议
- 注意在同一份合同书上签字可能被推定存在连带共同担保的意思联络

**3. 律师**
- 起草或审查担保合同时，明确担保人之间的关系
- 告知当事人担保人之间原则上不能相互追偿
- 引导当事人在合同中明确约定担保人之间的追偿权
- 计算担保人之间的分担比例

---

**适用场景**：共同担保、连带共同担保、混合担保、担保人追偿、担保人分担
