# 第17条：担保合同无效的责任分担

---

## 条文内容

> 主合同有效而第三人提供的担保合同无效，人民法院应当区分不同情形确定担保人的赔偿责任：
>
> （一）债权人与担保人均有过错的，担保人承担的赔偿责任不应超过债务人不能清偿部分的二分之一；
>
> （二）担保人有过错而债权人无过错的，担保人对债务人不能清偿的部分承担赔偿责任；
>
> （三）债权人有过错而担保人无过错的，担保人不承担赔偿责任。
>
> 主合同无效导致第三人提供的担保合同无效，担保人无过错的，不承担赔偿责任；担保人有过错的，其承担的赔偿责任不应超过债务人不能清偿部分的三分之一。

---

## 核心要点

### 1. 担保合同无效的两种情形

**情形1：主合同有效而担保合同无效**
- 担保合同因自身原因无效
- 与主合同的效力无关
- 典型原因：担保合同因欠缺有效要件而归于无效

**情形2：主合同无效导致担保合同无效**
- 基于担保的从属性
- 主合同无效 → 担保合同也无效

### 2. 担保人承担的责任性质：缔约过失责任

**性质**：
- 担保合同无效 → 担保人承担的是**缔约过失责任**
- 性质上属于**补充责任**
- 责任范围一般小于违约责任

**特点**：
- 担保人承担责任的范围统一限定为**债务人不能清偿的部分**
- 债务人是第一责任人
- 担保人是第二责任人

### 3. 主合同有效而担保合同无效的责任分担

#### 情形（一）：债权人与担保人均有过错

**基本规则**：担保人承担的赔偿责任不应超过债务人不能清偿部分的**二分之一**

**典型情形**：
- 法定代表人越权代表对外提供担保（《公司法》第16条）
- 债权人未进行合理审查（构成非善意）
- 担保人未履行内部决策程序

**示例**：
```
甲公司为乙公司向A银行借款100万元提供担保
甲公司的法定代表人越权签署担保合同
A银行未审查甲公司的股东会决议
→ 担保合同无效
→ A银行和甲公司均有过错
→ 甲公司承担的赔偿责任不应超过债务人不能清偿部分的1/2
```

#### 情形（二）：担保人有过错而债权人无过错

**基本规则**：担保人对债务人不能清偿的部分承担**赔偿责任**

**典型情形**：
- 担保人无民事行为能力
- 担保人违反法律禁止性规定（如公益机构担保）

**示例**：
```
A学校为乙公司向A银行借款100万元提供担保
A学校为公益法人
A银行明知学校不能提供担保仍接受担保
→ 担保合同无效
→ A学校有过错（明知不能提供担保仍提供）
→ A银行无过错
→ A学校对债务人不能清偿的部分承担赔偿责任
```

#### 情形（三）：债权人有过错而担保人无过错

**基本规则**：担保人**不承担赔偿责任**

**典型情形**：
- 债权人与债务人恶意串通骗取担保
- 债权人未告知担保人借新还旧事实

**示例**：
```
甲公司为乙公司向A银行借款100万元提供担保
A银行与乙公司恶意串通，骗取甲公司提供担保
→ 担保合同无效
→ A银行有过错
→ 甲公司无过错
→ 甲公司不承担赔偿责任
```

### 4. 主合同无效导致担保合同无效的责任分担

#### 担保人无过错

**基本规则**：担保人**不承担赔偿责任**

**典型情形**：
- 担保人对于主合同无效不知道或者不应当知道
- 担保人未促成主合同的成立

#### 担保人有过错

**基本规则**：担保人承担的赔偿责任不应超过债务人不能清偿部分的**三分之一**

**理由**：
- 主合同当事人（债权人和债务人）原则上也有过错
- 对于债权人的损失，应当以债权人、债务人、担保人**三方均分计算**为标准

**典型情形**：
- 担保人明知主合同无效仍为之提供担保
- 担保人明知主合同无效仍作为中介促成合同的订立

**示例**：
```
甲公司与乙公司签订买卖合同，约定甲公司向乙公司购买珍贵文物
丙公司为乙公司履行合同提供担保
后因买卖标的违法，买卖合同无效
→ 主合同无效 → 担保合同无效
→ 如果丙公司对于合同标的违法的事实不知情 → 丙公司无过错 → 不承担赔偿责任
→ 如果丙公司明知合同标的违法仍提供担保 → 丙公司有过错 → 承担不超过债务人不能清偿部分1/3的赔偿责任
```

## 深度实务要点

### 一、过错认定的具体标准

#### （一）债权人过错的认定

**典型情形**：

**1. 未履行合理审查义务**
- 公司对外担保：未审查公司决议（《公司法》第16条）
- 机关法人担保：明知机关法人不能提供担保仍接受担保
- 公益机构担保：明知公益机构不能提供担保仍接受担保

**2. 恶意串通**
- 债权人与债务人恶意串通骗取担保
- 故意隐瞒重要事实（如借新还旧）

**3. 违反诚实信用原则**
- 债权人明知担保合同无效仍接受担保
- 债权人利用担保人的无知或误解

#### （二）担保人过错的认定

**典型情形**：

**1. 违反法律禁止性规定**
- 机关法人提供担保（未经国务院批准）
- 公益机构提供担保（不符合除外情形）
- 无民事行为能力人或限制民事行为能力人提供担保

**2. 违反公司内部决策程序**
- 法定代表人越权代表对外提供担保
- 未履行董事会或股东会决议程序

**3. 明知主合同无效仍提供担保**
- 担保人明知主合同标的违法
- 担保人明知主合同双方恶意串通

**4. 其他过错情形**
- 担保人提供虚假材料
- 担保人故意隐瞒重要事实

#### （三）双方均有过错的认定

**典型情形**：

**1. 公司对外担保（越权代表）**
- 法定代表人越权签署担保合同（担保人过错）
- 债权人未审查公司决议（债权人过错）

**2. 借新还旧（担保人不同）**
- 债权人未告知担保人借新还旧事实（债权人过错）
- 担保人未进行合理审查（担保人过错，如金融机构）

**3. 其他混合过错情形**
- 双方均未尽合理审查义务
- 双方均对法律风险认识不足

---

### 二、"债务人不能清偿部分"的认定

#### 认定标准

**基本规则**：
- 债务人在债务到期后清偿债权的**剩余部分**
- 与债务人是否仍有清偿能力有关
- 如果债务人仍有清偿能力，则应当**先执行债务人的责任财产**

#### 认定程序

**Step 1：确定债权数额**
- 主债权本金
- 利息
- 违约金
- 损害赔偿金
- 实现债权的费用

**Step 2：确定债务人的清偿能力**
- 债务人的财产状况
- 债务人的现金流
- 债务人的其他负债情况

**Step 3：执行债务人的责任财产**
- 先执行债务人的财产
- 执行后仍不能清偿的部分 → 担保人承担责任

**Step 4：计算担保人的责任限额**
- 根据过错情形确定担保人的责任比例（1/2、1/3、100%）
- 担保人承担责任不超过责任限额

#### 示例计算

**示例1：双方均有过错**
```
债权数额：100万元
债务人清偿：40万元
债务人不能清偿部分：60万元
担保人责任限额：60万元 × 1/2 = 30万元
→ 担保人承担不超过30万元的赔偿责任
```

**示例2：担保人有过错而债权人无过错**
```
债权数额：100万元
债务人清偿：40万元
债务人不能清偿部分：60万元
担保人责任限额：60万元 × 100% = 60万元
→ 担保人承担60万元的赔偿责任
```

**示例3：主合同无效，担保人有过错**
```
债权数额：100万元
债务人清偿：40万元
债务人不能清偿部分：60万元
担保人责任限额：60万元 × 1/3 = 20万元
→ 担保人承担不超过20万元的赔偿责任
```

### 三、担保合同无效的常见情形与责任分担

#### 情形1：公司越权担保

**基本规则**：
- 法定代表人越权代表对外提供担保（《公司法》第16条）
- 债权人未审查公司决议 → 担保合同无效
- 债权人与担保人均有过错
- 担保人承担的赔偿责任不应超过债务人不能清偿部分的**1/2**

**示例**：
```
甲公司为乙公司向A银行借款100万元提供担保
甲公司的法定代表人越权签署担保合同
A银行未审查甲公司的股东会决议
→ 担保合同无效
→ A银行和甲公司均有过错
→ 甲公司承担的赔偿责任不应超过债务人不能清偿部分的1/2
```

#### 情形2：公益机构担保

**基本规则**：
- 以公益为目的的非营利法人、非法人组织提供担保
- 担保合同无效（除非符合除外情形）
- 债权人明知公益机构不能提供担保仍接受担保 → 债权人有过错
- 公益机构明知不能提供担保仍提供担保 → 担保人有过错

**示例1：债权人与担保人均有过错**
```
A学校为乙公司向A银行借款100万元提供担保
A学校为公益法人
A银行明知学校不能提供担保仍接受担保
A学校明知不能提供担保仍提供担保
→ 担保合同无效
→ A银行和A学校均有过错
→ A学校承担的赔偿责任不应超过债务人不能清偿部分的1/2
```

**示例2：担保人有过错而债权人无过错**
```
A医院为乙公司向A银行借款100万元提供担保
A医院为公益法人
A医院欺骗A银行，称其并非公益机构
A银行善意无过失
→ 担保合同无效
→ A医院有过错
→ A银行无过错
→ A医院对债务人不能清偿的部分承担赔偿责任
```

**示例3：债权人有过错而担保人无过错**
```
A学校为乙公司向A银行借款100万元提供担保
A学校为公益法人
A银行与乙公司恶意串通，骗取A学校提供担保
→ 担保合同无效
→ A银行有过错
→ A学校无过错
→ A学校不承担赔偿责任
```

#### 情形3：借新还旧（担保人不同）

**基本规则**：
- 新贷与旧贷的担保人不同
- 债权人未告知担保人借新还旧事实
- 担保人不知道借新还旧事实
- 担保合同无效
- 债权人有过错而担保人无过错
- 担保人不承担赔偿责任

**示例**：
```
A银行对乙公司享有100万元债权（旧贷）
丙公司为旧贷提供担保
A银行与乙公司协议借新还旧
A银行要求丁公司为新贷提供担保
A银行未告知丁公司借新还旧事实
丁公司不知道借新还旧事实
→ 新贷担保合同无效
→ A银行有过错
→ 丁公司无过错
→ 丁公司不承担赔偿责任
```

#### 情形4：主合同无效

**基本规则**：
- 主合同无效 → 担保合同无效（从属性）
- 担保人是否承担责任取决于其对主合同无效是否知情

**示例1：担保人无过错**
```
甲公司与乙公司签订买卖合同，约定甲公司向乙公司购买珍贵文物
丙公司为乙公司履行合同提供担保
后因买卖标的违法，买卖合同无效
→ 主合同无效 → 担保合同无效
→ 丙公司对合同标的违法的事实不知情 → 丙公司无过错
→ 丙公司不承担赔偿责任
```

**示例2：担保人有过错**
```
甲公司与乙公司签订买卖合同，约定甲公司向乙公司购买珍贵文物
丙公司为乙公司履行合同提供担保
丙公司明知合同标的违法仍提供担保
后因买卖标的违法，买卖合同无效
→ 主合同无效 → 担保合同无效
→ 丙公司明知合同标的违法仍提供担保 → 丙公司有过错
→ 丙公司承担的赔偿责任不应超过债务人不能清偿部分的1/3
```

### 四、实务争议与处理

#### 争议焦点1：担保人承担的是过错责任还是无过错责任？

**观点**：
- 主合同有效而担保合同无效：担保人承担的是**过错责任**
- 担保人是否承担责任，取决于其是否存在过错
- 如果担保人对于担保合同无效并无过错，不应由其承担民事责任

#### 争议焦点2：担保合同无效时，担保人能否向债务人追偿？

**观点**：
- 虽然担保合同无效，担保人基于过错承担民事赔偿责任
- 但该责任的承担仍是基于担保关系而产生
- 担保人承担的责任仍是代偿责任的性质
- 担保人有权在其承担赔偿责任的范围内向债务人追偿

**法律依据**：《民法典担保制度解释》第18条

#### 争议焦点3：担保合同无效时，担保人承担责任后能否向其他担保人追偿？

**观点**：
- 担保人原则上不得请求其他担保人承担担保责任
- 除非担保人之间存在相互追偿权（《民法典担保制度解释》第13条）

#### 争议焦点4：担保合同无效时，反担保合同的效力如何认定？

**观点**：
- 担保合同并非反担保合同的主合同
- 担保合同无效不必然导致反担保合同无效
- 反担保合同担保的是担保人对于债务人的追偿权
- 当事人仅以担保合同无效为由主张反担保合同无效的，人民法院不予支持

**法律依据**：《民法典担保制度解释》第19条

## 与旧法的区别

| 项目 | 旧法（担保法解释第7条） | 新法（民法典+本条） |
|------|----------------------|-------------------|
| 债权人无过错时的责任 | 担保人与债务人承担连带赔偿责任 | 担保人对债务人不能清偿的部分承担赔偿责任（补充责任） |
| 债权人有过错时担保人的责任 | 未明确规定 | 担保人不承担赔偿责任 |
| 责任性质 | 连带责任 | 补充责任 |
| 责任范围 | 债权人的经济损失 | 债务人不能清偿的部分 |

## 实务应用

### 案例1：公司越权担保，双方均有过错

**案情**：
甲公司为乙公司向A银行借款100万元提供担保。甲公司的法定代表人越权签署担保合同，未履行股东会决议程序。A银行未审查甲公司的股东会决议。后乙公司无力清偿债务，A银行起诉要求甲公司承担保证责任。甲公司抗辩称：担保合同无效，其不应承担担保责任。

**分析**：
1. 法定代表人越权代表对外提供担保 ✅
2. A银行未审查甲公司的股东会决议 ✅
3. 担保合同无效 ✅
4. A银行和甲公司均有过错 ✅
5. 根据《民法典担保制度解释》第17条第1款第1项：担保人承担的赔偿责任不应超过债务人不能清偿部分的1/2 ✅

**结论**：担保合同无效，甲公司承担的赔偿责任不应超过债务人不能清偿部分的1/2。

假设乙公司不能清偿60万元，甲公司承担不超过30万元的赔偿责任。

---

### 案例2：公益机构担保，担保人有过错而债权人无过错

**案情**：
A学校为乙公司向A银行借款100万元提供担保。A学校为公益法人，根据法律规定不得为他人提供担保。A银行善意无过失，不知道学校不能提供担保。后乙公司无力清偿债务，A银行起诉要求A学校承担保证责任。A学校抗辩称：其为公益法人，担保合同无效，其不应承担担保责任。

**分析**：
1. A学校为公益法人 ✅
2. 根据法律规定不得为他人提供担保 ✅
3. 担保合同无效 ✅
4. A银行善意无过失，不知道学校不能提供担保 → A银行无过错 ✅
5. A学校明知不能提供担保仍提供担保 → A学校有过错 ✅
6. 根据《民法典担保制度解释》第17条第1款第2项：担保人对债务人不能清偿的部分承担赔偿责任 ✅

**结论**：担保合同无效，A学校对债务人不能清偿的部分承担赔偿责任。

假设乙公司不能清偿60万元，A学校承担60万元的赔偿责任。

### 案例3：公益机构担保，债权人有过错而担保人无过错

**案情**：
A学校为乙公司向A银行借款100万元提供担保。A学校为公益法人，根据法律规定不得为他人提供担保。A银行明知学校不能提供担保仍接受担保。后乙公司无力清偿债务，A银行起诉要求A学校承担保证责任。A学校抗辩称：其为公益法人，担保合同无效，其不应承担担保责任。

**分析**：
1. A学校为公益法人 ✅
2. 根据法律规定不得为他人提供担保 ✅
3. 担保合同无效 ✅
4. A银行明知学校不能提供担保仍接受担保 → A银行有过错 ✅
5. A学校不知道其不能提供担保（如新成立的学校，对法律不了解）→ A学校无过错 ✅
6. 根据《民法典担保制度解释》第17条第1款第3项：担保人不承担赔偿责任 ✅

**结论**：担保合同无效，A学校不承担赔偿责任。A银行应自行承担放贷风险。

### 案例4：借新还旧，债权人有过错而担保人无过错

**案情**：
A银行对乙公司享有100万元债权（旧贷）。A银行与乙公司协议借新还旧，签订新的贷款合同，金额仍为100万元。A银行要求丙公司为新贷提供担保。A银行未告知丙公司借新还旧事实。丙公司不知道借新还旧事实。后乙公司无力清偿新贷，A银行起诉要求丙公司承担保证责任。丙公司抗辩称：A银行未告知借新还旧事实，其不应承担担保责任。

**分析**：
1. A银行与乙公司协议借新还旧 ✅
2. 新贷与旧贷的担保人不同（旧贷可能有其他担保人或无担保）✅
3. A银行未告知丙公司借新还旧事实 → A银行有过错 ✅
4. 丙公司不知道借新还旧事实 → 丙公司无过错 ✅
5. 根据《民法典担保制度解释》第16条第1款第2项：新贷与旧贷的担保人不同，人民法院不予支持 ✅
6. 根据《民法典担保制度解释》第17条第1款第3项：债权人有过错而担保人无过错的，担保人不承担赔偿责任 ✅

**结论**：新贷担保合同无效，丙公司不承担赔偿责任。A银行应自行承担放贷风险。

### 案例5：主合同无效，担保人有过错

**案情**：
甲公司与乙公司签订买卖合同，约定甲公司向乙公司购买珍贵文物，价格100万元。丙公司为乙公司履行合同提供担保。丙公司明知合同标的违法仍提供担保。后因买卖标的违法，买卖合同无效。甲公司起诉要求丙公司承担担保责任。丙公司抗辩称：主合同无效，担保合同也无效，其不应承担担保责任。

**分析**：
1. 买卖合同因标的违法无效 ✅
2. 主合同无效 → 担保合同无效 ✅
3. 丙公司明知合同标的违法仍提供担保 → 丙公司有过错 ✅
4. 根据《民法典担保制度解释》第17条第2款：担保人有过错的，其承担的赔偿责任不应超过债务人不能清偿部分的1/3 ✅

**结论**：担保合同无效，丙公司承担的赔偿责任不应超过债务人不能清偿部分的1/3。

假设乙公司不能清偿60万元，丙公司承担不超过20万元的赔偿责任。

---

## 争议焦点

**焦点1：担保人承担的是过错责任还是无过错责任？**

**观点**：
- 主合同有效而担保合同无效：担保人承担的是过错责任
- 担保人是否承担责任，取决于其是否存在过错

**焦点2：担保合同无效时，担保人能否向债务人追偿？**

**观点**：
- 担保人承担的责任仍是代偿责任的性质
- 担保人有权在其承担赔偿责任的范围内向债务人追偿

**焦点3：担保合同无效时，担保人承担责任后能否向其他担保人追偿？**

**观点**：
- 担保人原则上不得请求其他担保人承担担保责任
- 除非担保人之间存在相互追偿权

## 法律后果

### 主合同有效而担保合同无效

| 过错情形 | 担保人责任 |
|---------|-----------|
| 债权人与担保人均有过错 | 不超过债务人不能清偿部分的1/2 |
| 担保人有过错而债权人无过错 | 对债务人不能清偿的部分承担赔偿责任 |
| 债权人有过错而担保人无过错 | 不承担赔偿责任 |

### 主合同无效导致担保合同无效

| 过错情形 | 担保人责任 |
|---------|-----------|
| 担保人无过错 | 不承担赔偿责任 |
| 担保人有过错 | 不超过债务人不能清偿部分的1/3 |

## 相关条文

**《民法典》第682条第2款**：
保证合同被确认无效后，债务人、保证人、债权人有过错的，应当根据其过错各自承担相应的民事责任。

**《民法典》第388条第1款**：
担保合同是主债权债务合同的从合同。主债权债务合同无效的，担保合同无效，但是法律另有规定的除外。

**《民法典》第682条第2款**：
保证合同被确认无效后，债务人、保证人、债权人有过错的，应当根据其过错各自承担相应的民事责任。

**《民法典担保制度解释》第17条**：
主合同有效而第三人提供的担保合同无效，人民法院应当区分不同情形确定担保人的赔偿责任：
（一）债权人与担保人均有过错的，担保人承担的赔偿责任不应超过债务人不能清偿部分的二分之一；
（二）担保人有过错而债权人无过错的，担保人对债务人不能清偿的部分承担赔偿责任；
（三）债权人有过错而担保人无过错的，担保人不承担赔偿责任。

主合同无效导致第三人提供的担保合同无效，担保人无过错的，不承担赔偿责任；担保人有过错的，其承担的赔偿责任不应超过债务人不能清偿部分的三分之一。

**《民法典担保制度解释》第18条**：
承担了担保责任或者赔偿责任的担保人，在其承担责任的范围内向债务人追偿的，人民法院应予支持。

**《民法典担保制度解释》第19条**：
担保合同无效，承担了赔偿责任的担保人按照反担保合同的约定，在其承担赔偿责任的范围内请求反担保人承担担保责任的，人民法院应予支持。

## 实务建议

**1. 债权人（银行、金融机构等）**
- 审查担保人的担保资格（如公司决议、公益机构、机关法人等）
- 履行合理审查义务
- 告知担保人重要事实（如借新还旧）
- 避免与债务人恶意串通骗取担保

**2. 担保人**
- 审查自身的担保资格（如公司决策程序、法律禁止性规定）
- 了解主合同的内容（如是否借新还旧、合同标的是否合法）
- 避免明知不能提供担保仍提供担保
- 注意担保合同无效后仍可能承担赔偿责任

**3. 债务人**
- 注意主合同无效可能导致担保合同无效
- 担保合同无效后，债务人仍需承担还款责任
- 注意担保人承担赔偿责任后可能向债务人追偿

**4. 律师**
- 起草或审查担保合同时，审查担保人的担保资格
- 告知当事人担保合同无效的法律后果
- 分析当事人双方的过错程度
- 计算担保人的责任限额

---
