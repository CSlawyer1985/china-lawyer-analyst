# 第44条：主债权诉讼时效届满对担保物权的影响

---

## 条文内容

> 主债权诉讼时效期间届满后，抵押权人主张行使抵押权的，人民法院不予支持；抵押人以主债权诉讼时效期间届满为由，主张不承担担保责任的，人民法院应予支持。主债权诉讼时效期间届满前，债权人仅对债务人提起诉讼，经人民法院判决或者调解后未在民事诉讼法规定的申请执行时效期间内对债务人申请强制执行，其向抵押人主张行使抵押权的，人民法院不予支持。
>
> 主债权诉讼时效期间届满后，财产被留置的债务人或者对留置财产享有所有权的第三人请求债权人返还留置财产的，人民法院不予支持；债务人或者第三人请求拍卖、变卖留置财产并以所得价款清偿债务的，人民法院应予支持。
>
> 主债权诉讼时效期间届满的法律后果，以登记作为公示方式的权利质权，参照适用第一款的规定；动产质权、以交付权利凭证作为公示方式的权利质权，参照适用第二款的规定。

---

## 核心要点

### 1. 基本原则：主债权诉讼时效届满 → 担保物权消灭

**抵押权**：
- 主债权诉讼时效期间届满后，抵押权人主张行使抵押权的 → 人民法院不予支持
- 抵押人以主债权诉讼时效期间届满为由，主张不承担担保责任的 → 人民法院应予支持

**留置权**：
- 主债权诉讼时效期间届满后，财产被留置的债务人或者对留置财产享有所有权的第三人请求债权人返还留置财产的 → 人民法院不予支持
- 债务人或者第三人请求拍卖、变卖留置财产并以所得价款清偿债务的 → 人民法院应予支持

**质权**：
- 以登记作为公示方式的权利质权 → 参照适用抵押权的规定
- 动产质权、以交付权利凭证作为公示方式的权利质权 → 参照适用留置权的规定

### 2. 立法目的

**立法目的**：
- 维护诉讼时效制度的权威性
- 避免担保物权长期存在影响物的流通
- 平衡债权人和担保人的利益

---

## 深度实务要点

### 一、主债权诉讼时效届满对抵押权的影响

#### 基本规则

**主债权诉讼时效期间届满后**：
- 抵押权人主张行使抵押权的 → 人民法院不予支持
- 抵押权消灭
- 抵押人有权要求注销抵押登记

**示例**：
```
主债权：100万元
主债权诉讼时效期间：3年（自主债权履行期限届满之日起计算）
主债权履行期限届满之日：2023年1月1日
主债权诉讼时效期间届满之日：2026年1月1日
抵押权人于2026年2月1日主张行使抵押权 → 人民法院不予支持
```

#### 特殊情形：债权人仅对债务人提起诉讼但未申请执行

**基本规则**：
- 主债权诉讼时效期间届满前，债权人仅对债务人提起诉讼
- 经人民法院判决或者调解后未在民事诉讼法规定的申请执行时效期间内对债务人申请强制执行
- 其向抵押人主张行使抵押权的 → 人民法院不予支持

**申请执行时效期间**：
- 2年（自法律文书规定履行期限的最后一日起计算）
- 适用中止、中断、延长的规定

**示例**：
```
主债权履行期限届满之日：2023年1月1日
债权人仅对债务人提起诉讼：2025年6月1日（诉讼时效内）
人民法院判决：2025年9月1日，判决债务人于判决生效后10日内履行
判决生效之日：2025年10月1日
申请执行时效期间：自2025年10月11日起2年 → 2027年10月11日
债权人于2027年11月1日申请强制执行 → 超过申请执行时效期间
→ 债权人向抵押人主张行使抵押权 → 人民法院不予支持
```

---

### 二、主债权诉讼时效届满对留置权的影响

#### 基本规则

**主债权诉讼时效期间届满后**：

**情形1：债务人或者第三人请求返还留置财产**
- 请求债权人返还留置财产的 → 人民法院不予支持

**情形2：债务人或者第三人请求拍卖、变卖留置财产**
- 请求拍卖、变卖留置财产并以所得价款清偿债务的 → 人民法院应予支持

**理由**：
- 留置权具有优先受偿性
- 留置权人占有留置财产，债务人不履行债务时，留置权人有权优先受偿
- 但主债权诉讼时效届满后，主债权成为自然债务
- 留置权人不能继续占有留置财产，但可以行使优先受偿权

---

### 三、主债权诉讼时效届满对质权的影响

#### 基本规则

**以登记作为公示方式的权利质权**：
- 参照适用抵押权的规定
- 主债权诉讼时效期间届满后，质权人主张行使质权的 → 人民法院不予支持

**动产质权、以交付权利凭证作为公示方式的权利质权**：
- 参照适用留置权的规定
- 主债权诉讼时效期间届满后：
  - 出质人请求返还质物的 → 人民法院不予支持
  - 出质人请求拍卖、变卖质物并以所得价款清偿债务的 → 人民法院应予支持

---

## 实务应用

### 案例1：主债权诉讼时效届满，抵押权消灭

**案情**：
甲公司向A银行借款100万元，借款期限为1年（2022年1月1日至2022年12月31日）。乙公司以其房产为甲公司的债务提供抵押担保，并办理抵押登记。后甲公司未按期还款，A银行未向甲公司主张权利。2025年2月1日，A银行起诉要求乙公司承担抵押担保责任。乙公司抗辩称：主债权诉讼时效已届满，抵押权已消灭。

**分析**：
1. 主债权履行期限届满之日：2022年12月31日 ✅
2. 主债权诉讼时效期间：3年（自2022年12月31日起至2025年12月31日）✅
3. A银行于2025年2月1日起诉 → 诉讼时效内 ✅
4. A银行主张行使抵押权 → 人民法院应予支持 ✅

**结论**：A银行在诉讼时效内起诉，抵押权未消灭，A银行有权行使抵押权。

---

### 案例2：债权人仅对债务人提起诉讼但未申请执行

**案情**：
甲公司向A银行借款100万元，借款期限为1年（2022年1月1日至2022年12月31日）。乙公司以其房产为甲公司的债务提供抵押担保，并办理抵押登记。后甲公司未按期还款，A银行于2025年6月1日仅起诉甲公司，人民法院于2025年9月1日判决甲公司于判决生效后10日内履行。判决生效后，A银行未在申请执行时效期间内申请强制执行。2027年11月1日，A银行起诉要求乙公司承担抵押担保责任。乙公司抗辩称：A银行未在申请执行时效期间内申请强制执行，抵押权已消灭。

**分析**：
1. A银行于2025年6月1日仅起诉甲公司 ✅
2. 人民法院于2025年9月1日判决甲公司于判决生效后10日内履行 ✅
3. 判决生效之日：2025年10月1日 ✅
4. 申请执行时效期间：自2025年10月11日起2年 → 2027年10月11日 ✅
5. A银行于2027年11月1日申请强制执行 → 超过申请执行时效期间 ✅
6. 根据《民法典担保制度解释》第44条第1款后段：其向抵押人主张行使抵押权的，人民法院不予支持 ✅

**结论**：A银行未在申请执行时效期间内申请强制执行，抵押权消灭，A银行无权行使抵押权。

---

### 案例3：主债权诉讼时效届满对留置权的影响

**案情**：
甲公司将汽车送至乙公司维修，维修费1万元。乙公司修好汽车后，甲公司未支付维修费。乙公司依法留置该汽车。后乙公司一直未向甲公司主张权利。3年诉讼时效期间届满后，甲公司请求乙公司返还汽车。

**分析**：
1. 主债权：维修费1万元 ✅
2. 主债权诉讼时效期间：3年 ✅
3. 主债权诉讼时效期间届满 ✅
4. 根据《民法典担保制度解释》第44条第2款：财产被留置的债务人请求债权人返还留置财产的，人民法院不予支持 ✅

**结论**：乙公司无需返还汽车，但甲公司可以请求拍卖、变卖留置财产并以所得价款清偿债务。

## 争议焦点

**焦点1：主债权诉讼时效届满，担保物权是否消灭？**

**观点**：
- 抵押权和以登记作为公示方式的权利质权：主债权诉讼时效届满后，担保物权消灭
- 留置权、动产质权和以交付权利凭证作为公示方式的权利质权：主债权诉讼时效届满后，担保权人不能继续占有担保财产，但可以行使优先受偿权

**焦点2：债权人仅对债务人提起诉讼但未申请执行，抵押权是否消灭？**

**观点**：
- 债权人仅对债务人提起诉讼，经人民法院判决或者调解后未在申请执行时效期间内对债务人申请强制执行
- 其向抵押人主张行使抵押权的 → 人民法院不予支持

## 法律后果

### 抵押权、以登记作为公示方式的权利质权

**基本规则**：
- 主债权诉讼时效期间届满 → 担保物权消灭
- 担保物权人主张行使担保物权的 → 人民法院不予支持
- 担保人主张不承担担保责任的 → 人民法院应予支持

### 留置权、动产质权、以交付权利凭证作为公示方式的权利质权

**基本规则**：
- 主债权诉讼时效期间届满 → 担保权人不能继续占有担保财产
- 出质人请求返还质物的 → 人民法院不予支持
- 出质人请求拍卖、变卖质物并以所得价款清偿债务的 → 人民法院应予支持

## 相关条文

**《民法典》第419条**：
抵押权人应当在主债权诉讼时效期间行使抵押权；未行使的，人民法院不予保护。

**《民法典》第437条**：
出质人请求质权人及时行使质权，因质权人怠于行使权利造成出质人损害的，由质权人承担赔偿责任。

**《民法典》第452条**：
留置权人与债务人应当约定留置财产后的债务履行期限；没有约定或者约定不明确的，留置权人应当给债务人六十日以上履行债务的期限，但是鲜活易腐等不易保管的动产除外。债务人逾期未履行的，留置权人可以与债务人协议以留置财产折价，也可以就拍卖、变卖留置财产所得的价款优先受偿。

## 实务建议

**1. 债权人（银行、金融机构等）**
- 注意主债权诉讼时效期间为3年
- 及时向债务人和担保人主张权利
- 及时对债务人提起诉讼或者申请仲裁
- 及时申请强制执行
- 在主债权诉讼时效期间内行使担保物权

**2. 抵押人、出质人**
- 注意主债权诉讼时效期间
- 主债权诉讼时效期间届满后，可以主张不承担担保责任
- 对于抵押权和以登记作为公示方式的权利质权，可以要求注销登记

**3. 律师**
- 告知债权人在主债权诉讼时效期间内行使权利
- 告知担保人主债权诉讼时效届满的法律后果
- 审查债权人是否在主债权诉讼时效期间内行使权利
