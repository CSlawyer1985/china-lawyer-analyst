# 第65条：违约金调整标准

---

## 条文内容

> 当事人主张约定的违约金过分高于违约造成的损失，请求予以适当减少的，人民法院应当以民法典第五百八十四条规定的损失为基础，兼顾合同主体、交易类型、合同的履行情况、当事人的过错程度、履约背景等因素，遵循公平原则和诚信原则予以衡量。
>
> 约定的违约金超过造成损失的百分之三十的，人民法院可以认定为民法典第五百八十四条规定的"过分高于造成的损失"。
>
> 恶意违约的当事人一方请求减少违约金的，人民法院一般不予支持。

---

## 核心要点

### 1. 调整基准：民法典第584条

**损失范围**：
- 实际损失
- 可得利益损失

**限制**：不得超过违约一方订立合同时预见到或者应当预见到的因违约可能造成的损失。

### 2. 调整因素（综合考虑）

| 因素 | 说明 |
|------|------|
| 合同主体 | 自然人、法人、专业主体等 |
| 交易类型 | 买卖、借款、租赁等 |
| 合同履行情况 | 部分履行、全部未履行等 |
| 当事人过错程度 | 故意、重大过失、一般过失 |
| 履约背景 | 商业谈判地位、信息对称性等 |

### 3. 调整标准：30%规则

**一般规则**：
```
约定的违约金 > 造成损失 × 130% → 一般认定为"过分高于造成的损失"
```

**示例**：
- 实际损失：100万元
- 违约金上限：130万元
- 约定违约金：200万元
- → 可以请求调整至130万元

### 4. 恶意违约的惩罚

**恶意违约的认定**：
- 故意违约
- 违背诚信原则
- 恶意磋商、恶意毁约

**后果**：当事人一方请求减少违约金 → 人民法院一般不予支持

## 深度实务要点（基于理解与适用）

### 一、30%标准的适用和例外

#### 一般规则

**基本原则**：约定的违约金超过造成损失的30% → 人民法院可以认定为"过分高于造成的损失"

**计算公式**：
```
约定的违约金 > 造成损失 × 130% → 一般认定为"过分高于造成的损失"
```

**示例**：
- 实际损失：100万元
- 违约金上限：130万元（100万 × 130%）
- 约定违约金：200万元
- → 可以请求调整至130万元

#### 30%规则是否绝对？

**不是绝对标准**：30%是"一般认定"标准，法院可以根据具体情况调整

**允许更高违约金的情形**：
- 交易风险较高的合同
- 专业主体之间的商业合同
- 当事人有明确约定且不违反法律强制性规定

**适用更严格标准的情形**：
- 格式合同
- 弱势一方参与的合同
- 消费者合同

---

### 二、举证责任的动态移转机制（核心要点）

#### 1. 客观证明责任（立法者在实体法中分配）

**基本规则**：
- 主张违约金过高的当事人 → 承担举证证明责任
- 主张违约金低于实际损失的当事人 → 承担举证证明责任

**理由**：
- 违约金的性质：**损失预定**
- 违约金的功能：**分配证明责任**

#### 2. 主观证明责任（法官在具体案件中分配）

**核心规则**：**动态移转机制**

**阶段1：违约方初步举证**
- 违约方举出**初步证据**
- 使法官产生**合理怀疑**
- 证据标准：**低**（初步证据即可）

**阶段2：证明责任移转给守约方**
- 守约方需**推翻"合理怀疑"**
- 证据标准：**高度可能性**

**阶段3：法官自由心证**
- 如果守约方不能推翻"合理怀疑"
- 法官即可产生违约金过高的内心确信

#### 3. 为什么采用动态移转机制？

**核心理由**：
- 损失发生在守约方
- 让违约方举证证明守约方的损失且达到高度可能性 → **实属难为**

**示例**：
```
违约方主张：违约金过高
  ↓
举证1：提供初步证据（如市场价格下降）
  ↓
法官：产生"合理怀疑"
  ↓
移转给守约方
  ↓
守约方需推翻"合理怀疑"：
  - 证明实际损失更高
  - 证明违约金并非过高
  - 证明存在特殊情形（如高风险交易）
  ↓
如果守约方不能举证推翻：
  → 法官即可产生违约金过高的内心确信
```

---

### 三、"合理怀疑"的判断标准

#### 判断标准（法官自由心证）

**典型情形**：
1. **约定违约金明显高于市场惯例**
   - 同类交易通常违约金为合同总额的5-10%
   - 当事人约定30%或更高

2. **实际损失明显低于约定违约金**
   - 实际损失仅占约定违约金的小部分
   - 违约方提供初步证据证明损失不大

3. **合同履行比例与违约金不成比例**
   - 仅轻微违约，但约定全额违约金
   - 部分履行，但约定按全部未履行计算违约金

4. **违约方因违约获得的利益明显小于违约金**
   - 违约方违约并非为了获得更大利益
   - 违约金明显具有惩罚性而非补偿性

**"合理怀疑"的程度**：
- 不是"高度可能性"
- 也不是"优势证据"
- 而是**足以引起法官疑问的初步证据**

---

### 四、酌定损失赔偿额的6个考量因素

#### 适用情形

当难以确定可得利益时，人民法院应当综合考虑以下因素：

**1. 违约方因违约获得的利益**
- 违约方是否通过违约获得利益
- 获得的利益大小
- 利益是否为违约的主要动机

**2. 违约方的过错程度**
- 故意违约
- 重大过失
- 一般过失
- 轻微过失

**3. 其他违约情节**
- 违约的方式、手段
- 违约后的态度（是否积极补救）
- 是否恶意磋商、恶意毁约
- 是否一物二卖、恶意转让财产

**4. 合同主体**
- 自然人 vs 法人
- 专业主体 vs 普通主体
- 商事主体 vs 消费者

**5. 交易类型**
- 买卖合同
- 借款合同
- 租赁合同
- 建设工程合同等

**6. 履行背景**
- 市场价格波动情况
- 政策变化影响
- 不可抗力因素

**遵循原则**：
- **公平原则**
- **诚信原则**

---

### 五、恶意违约的认定与后果

#### 恶意违约的认定标准

**1. 明确表示不履行合同**
- 无正当理由拒绝履行
- 明确表示将不履行合同义务

**2. 履行能力严重不足仍签订合同**
- 明知无法履行仍签订合同
- 恶意签订无法履行的合同

**3. 恶意磋商、恶意毁约**
- 假借磋商恶意拖延时间
- 恶意毁约导致对方损失扩大

**4. 一物二卖、恶意转让财产**
- 为获取更高价款而将同一标的物卖给第三人
- 恶意转让财产以逃避履行义务

#### 后果：一般不予支持减少违约金

**基本规则**：
- 恶意违约的当事人一方请求减少违约金 → 人民法院一般不予支持

**理由**：
- 惩罚恶意违约行为
- 维护诚信原则
- 保护守约方利益

**"一般不予支持"的含义**：
- 不是绝对不支持
- 例外情形：违约金确实过分夸张（如超过损失的500%）
- 法院仍可根据公平原则予以适当减少

## 与旧法的区别

| 项目 | 旧法（《合同法》第114条第2款） | 新法（本条） |
|------|------------------------------|-------------|
| 调整标准 | "过分高于"（不明确） | 超过损失30% |
| 调整因素 | 未规定 | 明确5个因素 |
| 恶意违约 | 未规定 | 一般不予支持 |
| 裁判标准 | 自由裁量 | 明确量化标准 |

## 实务应用

### 案例：高额违约金的调整

**案情**：合同约定违约金为合同总额的50%，违约方主张过分高于实际损失。

**分析**：
- 根据第65条第2款，约定的违约金超过造成损失的30% → 一般认定为过分高于造成的损失
- 以民法典第584条规定的损失为基础（实际损失+可得利益损失）
- 兼顾合同主体、交易类型、合同的履行情况、当事人的过错程度、履约背景等因素
- 遵循公平原则和诚信原则

**示例计算**：
```
合同总额：1000万元
约定违约金：500万元（50%）
实际损失：200万元
损失130%：260万元
→ 可以调整至260万元

如果违约方恶意违约：
→ 一般不予支持减少
→ 维持500万元违约金
```

**结论**：一般违约情形下调至260万元；恶意违约情形下维持500万元。

## 争议焦点

**焦点1**：30%规则是否绝对？

**观点**：30%是"一般认定"标准，不是绝对标准。法院可以根据具体情况调整：
- 交易风险较高的合同 → 可能允许更高的违约金
- 格式合同、弱势一方 → 可能适用更严格的标准

**焦点2**："恶意违约"的认定标准？

**观点**：
- 明确表示不履行合同
- 履行能力严重不足仍签订合同
- 恶意磋商、恶意毁约
- 一物二卖、恶意转让财产

**焦点3**：实际损失难以证明时如何处理？

**观点**：
- 依据第62条，综合考虑违约方因违约获得的利益、违约方的过错程度、其他违约情节等因素
- 遵循公平原则和诚信原则确定
- 可以参照合同总额的一定比例（通常20%-30%）
